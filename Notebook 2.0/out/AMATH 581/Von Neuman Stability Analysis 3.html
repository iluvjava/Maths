Time Stepping Method

Assuming a general format for the PDEs in the following form:

ut=f(∂x2u,∂xu)u_t = f(\partial_x^2u, \partial_xu) ut​=f(∂x2​u,∂x​u)

So it's second order, non time depend, but doesn't have to be linear, then we define the CFL numbers like the following:

λ=ΔtΔx\lambda = \frac{\Delta t}{\Delta x} λ=ΔxΔt​

Forward Euler

un(m+1)=un(m)+f(un+1(m)−un−1(m)2Δx,un+1(m)−2un(m)+un−1(m)Δx2)u_n^{(m + 1)} = u_n^{(m)} + f\left( \frac{u_{n + 1}^{(m)}- u_{n - 1}^{(m)}}{2\Delta x}, \frac{u_{n + 1}^{(m )} - 2u_{n}^{(m )} + u_{n - 1}^{(m)}} {\Delta x^2} \right) un(m+1)​=un(m)​+f(2Δxun+1(m)​−un−1(m)​​,Δx2un+1(m)​−2un(m)​+un−1(m)​​)

If fff is linear, then this is easier to analyze and can be put into a linear system for time stepping.

Order: O(Δx2Δt)\mathcal{O}(\Delta x^2\Delta t)O(Δx2Δt)

  • Wave Equation: Unconditionally Unstable
  • Heat Equation:

    • CΔtΔx2<12\frac{C\Delta t}{\Delta x^2} < \frac{1}{2} Δx2CΔt​<21​
