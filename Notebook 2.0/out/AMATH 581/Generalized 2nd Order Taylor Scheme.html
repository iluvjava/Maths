Here we will discuss a class of time-stepping scheme found using the Taylor Series expansion of the Ordinary Differential Equations. prereq: [[Runge Kutta]] This is important to understand what this is doing here.

Let's present the following formula for our time stepping scheme:

y(t+h)=y(t)+h[Af(t,y(t))+Bf(t+Ph,y(t)+Qhf(t,y(t))]y(t + h) = y(t) + h\left[Af(t, y(t)) + Bf(t+Ph, y(t) + Qhf(t, y(t))\right] y(t+h)=y(t)+h[Af(t,y(t))+Bf(t+Ph,y(t)+Qhf(t,y(t))]

Notice that the nested function inside of it can be approximated with a first order taylor series for function with 2 variables (Although in this case the special dimension is a huge vector), And the h is just Δt\Delta tΔt.

Here we simply write yyy as y(t)y(t)y(t)

f(t+Ph,y(t)+Qhf(t,y))=f(t,y)+Phft(t,y)+Qhf(t,y)fy(t,y)f(t + Ph, y(t) + Qhf(t, y)) = f(t, y) + Phf_t(t, y) + Qhf(t, y)f_y(t, y) f(t+Ph,y(t)+Qhf(t,y))=f(t,y)+Phft​(t,y)+Qhf(t,y)fy​(t,y)

And then we got the following formula that frame the variables A,B,P,QA, B, P, QA,B,P,Q, giving us:

Heun's Method

y(t+h)=y+h2(f(t,y)+f(t+h,y+hf(t,y)))y(t + h) = y + \frac{h}{2}(f(t, y) + f(t + h, y + hf(t, y))) y(t+h)=y+2h​(f(t,y)+f(t+h,y+hf(t,y)))

This is when A=B=12A = B = \frac{1}{2}A=B=21​ and P=Q=1P = Q = 1P=Q=1

This is a method with local accuracy of O(h3)\mathcal{O}(h^3)O(h3) and a quadratic global accuracy.

Note: This scheme is the same as the second order Runge Kutta method.
