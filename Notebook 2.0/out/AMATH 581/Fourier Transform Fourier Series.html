Properties of the Fourier Transform

Derivative is a Linear transform in the Fourier Domain.

f(n)(x)^=(ik)nf(x)^\widehat{f^{(n)}(x)} = (ik)^n\widehat{f(x)} f(n)(x)​=(ik)nf(x)​

Say we want to solve the equation:

y′′(x)−ω2y(x)=−f(x)y''(x) - \omega^2y(x) = -f(x) y′′(x)−ω2y(x)=−f(x)

Where x∈[−∞,∞]x\in [-\infty, \infty]x∈[−∞,∞]

And then applying the Fourier Transform to both side of the equations we are going to have:

−k2y^−ω2y^=−f^-k^2\widehat{y} - \omega^2\widehat{y} = -\widehat{f} −k2y​−ω2y​=−f​

Solving for yyy, we are going to have this:

y^=f^k2+ω2\widehat{y} = \frac{\widehat{f}}{k^2 + \omega^2} y​=k2+ω2f​​

And then, we are literally one step away from getting the solution, applying an inverse Fourier Transform here to get the expression for the quantity y^\widehat{y}y​.

NOTE:

This derivative property only applies if the function is absolutely integrable on the whole R\mathbb{R}R domain, meaning that the absolute value of the function is finite over the whole real realm.

Convolution Theorem

F(fg∣k)=F(f∣k)∗F(g∣k)\mathcal{F}(fg|k) = \mathcal{F}(f|k) * \mathcal{F}(g|k) F(fg∣k)=F(f∣k)∗F(g∣k)

where * is the convolution operator where it's defined like:

(f∗g)(x)=∫−∞∞f(y)g(x−y)dy(f*g)(x) = \int_{-\infty}^{\infty} f(y)g(x - y)dy (f∗g)(x)=∫−∞∞​f(y)g(x−y)dy

The Fourier Transform of the product of 2 function is the convolution of those 2 functions under the Fourier Domain!

And this is really convenient for Transforming Non-Linear PDEs.

See Convolution For more information about the theorem.

And this is used for spectral method when there are non-linear terms involved in the equation.

Pros and Cons DFT

  1. Operations is O(Nlog⁡(N))\mathcal{O}(N \log(N))O(Nlog(N)), for uniformly discretized spatial dimension. Other topology are not applicable for DFT.
  2. It's only suitable for Periodic Conditions, if not the boundary will be look really ugly and there will be oscillations. (And in that case please consult prof Kutz book on Scientific Computing, where 2 pinned boundary conditions is discussed for the spectral methods)
  3. Discretization has to be 2n2^n2n.
  4. The order of accuracy of the method is exponential, so it's pretty insane.
