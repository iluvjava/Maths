Von-Neumann Stability Analysis

Assume that the solution is separable, and using separation of variables we have:

u(x,t)=G(t)F(x)u(x, t) = G(t)F(x) u(x,t)=G(t)F(x)

The key here is to analyze a solution of this form:

un(m)=∑k=−∞∞gmckexp⁡(ikπxnL)u_n^{(m)} = \sum_{k = -\infty}^\infty g^mc_k\exp \left( \frac{ik\pi x_n}{L} \right) un(m)​=k=−∞∑∞​gmck​exp(Likπxn​​)

This is like the classic solution to elliptic PDEs, and under this case, all the Fourier Coefficients will cancel out after plugging them into the scheme, simultaneously, all the trigs with the same frequencies can be collected together, giving us a really nice form for stability analysis.

Stability Criterion

∣g∣<1|g| < 1∣g∣<1 --> Stable ∣g∣≤1|g| \leq 1∣g∣≤1 --> Unstable

So, we are basically investigating how the wave changes if we were to use some kind of scheme. It's very much similar to stability analysis of the ODEs equations, where we use the exponential model for the analysis.

FE Scheme on Wave Equation

So the key here is to analyze the stability is to substitute the Fourier Series into the scheme and see how the multiplier grows. If it explodes, then the scheme is going to be unstable.

The FE is:

un(m+1)=un(m)+λ2(un+1m−un−1m)u^{(m + 1)}_n = u^{(m)}_n + \frac{\lambda}{2}(u^{m}_{n + 1} - u_{n - 1}^m) un(m+1)​=un(m)​+2λ​(un+1m​−un−1m​)

And now, let's focus on the Fourier Series representation, and it will be like:

un(m)=∑k=−∞∞gmckexp⁡(ikπxnL)u_n^{(m)} = \sum_{k = -\infty}^\infty g^mc_k\exp \left( \frac{ik\pi x_n}{L} \right) un(m)​=k=−∞∑∞​gmck​exp(Likπxn​​)

substituting it, collect the like terms, and cancel out the Fourier Coefficients, we will have:

∀k∈Z:gm+1exp⁡(ikπxnL)=gmexp⁡(ikπxnL)+λ2(gmexp⁡(ikπxn+1L)−gmexp⁡(ikπxn−1L))\forall k \in \mathbb{Z}: \quad g^{m + 1}\exp \left( \frac{ik\pi x_n}{L} \right) = g^m\exp\left( \frac{ik\pi x_n}{L} \right) + \frac{\lambda}{2} \left( g^m\exp \left( \frac{ik\pi x_{n + 1}}{L} \right) - g^m\exp\left( \frac{ik\pi x_{n - 1}}{L} \right) \right) ∀k∈Z:gm+1exp(Likπxn​​)=gmexp(Likπxn​​)+2λ​(gmexp(Likπxn+1​​)−gmexp(Likπxn−1​​))

And then we make it look better using:

ζn=ikπxnL\zeta_n = \frac{ik\pi x_n}{L} ζn​=Likπxn​​

So we have it put into a nicer form and it's like:

gm+1exp⁡(ζn)=gmexp⁡(ζn)+λ2(gmexp⁡(ζn+1)−gmexp⁡(ζn−1))g^{m + 1} \exp\left( \zeta_n \right) = g^m \exp\left( \zeta_n \right) + \frac{\lambda}{2} \left( g^m\exp(\zeta_{n + 1}) - g^m\exp(\zeta_{n - 1}) \right) gm+1exp(ζn​)=gmexp(ζn​)+2λ​(gmexp(ζn+1​)−gmexp(ζn−1​))

Let's divide both side by this quantity: gmexp⁡(ζn)g^m\exp(\zeta_n)gmexp(ζn​), which should give us:

g=1+λ2(exp⁡(ζn+1−ζn)−exp⁡(ζn−1−ζn))g = 1 + \frac{\lambda}{2}\left( \exp(\zeta_{n + 1} - \zeta_n) - \exp(\zeta_{n - 1} - \zeta_{n}) \right) g=1+2λ​(exp(ζn+1​−ζn​)−exp(ζn−1​−ζn​))

Note that ζn+1−ζn=ikπΔxL\zeta_{n + 1} - \zeta_{n} = \frac{ik\pi\Delta x}{L}ζn+1​−ζn​=LikπΔx​ denote this as Δξ\Delta\xiΔξ

then we have:

g=1+λ2(exp⁡(Δξ)−exp⁡(−Δξ))g = 1 + \frac{\lambda}{2}(\exp(\Delta\xi) - \exp(-\Delta \xi)) g=1+2λ​(exp(Δξ)−exp(−Δξ))

Which is:

g=1+2iλ2sin⁡(Δξ)g = 1 + \frac{2i\lambda}{2}\sin(\Delta \xi) g=1+22iλ​sin(Δξ)

And notice that $$|g|^2 > 1$$ will be true regardless of what we choose for λ\lambdaλ, there fore this scheme is always going to be unstable.

Backwards Euler on Wave Equation

This is what the backward Euler Scheme looks like for the wave equation with Temporal Discretization:

un(m+1)=un(m)+λ2(un+1(m+1)−un−1(m+1))u_n^{(m + 1)} = u_n^{(m)} + \frac{\lambda}{2}{(u_{n + 1}^{(m + 1)} - u_{n - 1}^{(m + 1)}}) un(m+1)​=un(m)​+2λ​(un+1(m+1)​−un−1(m+1)​)

Where we need to solve for the future time step un(m+1)u_n^{(m + 1)}un(m+1)​

And this will give us this when we substitute the Fourier Series for Analysis:

gm+1exp⁡(ζn)=gmexp⁡(ζn)+λ2(gm+1exp⁡(ζn+1)−gm+1exp⁡(ζn−1))g^{m + 1}\exp(\zeta_n) = g^{m}\exp(\zeta_{n}) + \frac{\lambda}{2}( g^{m + 1}\exp(\zeta_{n + 1}) - g^{m + 1}\exp(\zeta_{n - 1})) gm+1exp(ζn​)=gmexp(ζn​)+2λ​(gm+1exp(ζn+1​)−gm+1exp(ζn−1​))

And diving both side by gmexp⁡(ζn)g^{m}\exp(\zeta_n)gmexp(ζn​) and then the right hand side will be ggg, then we have:

g=1+λ2(gexp⁡(−Δξ)−gexp⁡(Δξ))g = 1 + \frac{\lambda}{2}( g\exp(-\Delta \xi) - g\exp(\Delta \xi) ) g=1+2λ​(gexp(−Δξ)−gexp(Δξ))

g=1+gλ2(exp⁡(−Δξ)−exp⁡(Δξ))g = 1 + \frac{g\lambda}{2}( \exp(-\Delta \xi) - \exp(\Delta \xi) ) g=1+2gλ​(exp(−Δξ)−exp(Δξ))

g=1−igλsin⁡(Δξ)g = 1 - ig\lambda\sin(\Delta \xi) g=1−igλsin(Δξ)

Which means that:

g=11+iλsin⁡(Δξ)g = \frac{1}{1 + i\lambda\sin(\Delta\xi)} g=1+iλsin(Δξ)1​

Notice that, this is** unconditionally stable**, because:

∣g∣2=11+λ2sin⁡2(Δξ)≤1∀  k,λ|g|^2 = \frac{1}{1+ \lambda^2\sin^2(\Delta \xi)} \leq 1 \quad\forall\; k, \lambda ∣g∣2=1+λ2sin2(Δξ)1​≤1∀k,λ

Leap Frog on Wave Equation (More Common)

Skipping the math part, here is the scheme for leapfrog method on the wave equation:

un(m+1)=un(m−1)+λ(un+1(m)−un−1(m))u_n^{(m + 1)} = u_n^{(m - 1)} + \lambda(u_{n + 1}^{(m)} - u_{n - 1}^{(m)}) un(m+1)​=un(m−1)​+λ(un+1(m)​−un−1(m)​)

And after some simplification, we will have:

g2=1+ig2λsin⁡(Δξ)g^2 = 1 + ig2\lambda\sin(\Delta \xi) g2=1+ig2λsin(Δξ)

This is an equation we need to solve for, and solving it should give us:

g=iλsin⁡(Δξ)±−λ2sin⁡2(Δξ)−1g = i\lambda\sin(\Delta\xi) \pm \sqrt{-\lambda^2\sin^2(\Delta \xi) - 1} g=iλsin(Δξ)±−λ2sin2(Δξ)−1​

Consider ∣g∣<1|g| < 1∣g∣<1, and we consider the worse case scenario for this equation containing multiple solutions.

And we will have ∣g∣<1|g| < 1∣g∣<1 for λ<λmax=1\lambda < \lambda_{max} = 1λ<λmax​=1

Then we will need to have the criteria that:

λ=CΔtΔx<1  ⟹  Δt<ΔxC\lambda = \frac{C\Delta t}{\Delta x} < 1 \implies \Delta t < \frac{\Delta x}{C} λ=ΔxCΔt​<1⟹Δt<CΔx​

And this is the criteria for Δt\Delta tΔt, we have to change the Δt\Delta tΔt whenever we are given a space discretization

Comment, Summary

The FE for 1d Wave equation is unconditionally unstable. The BE for 1d wave equation is unconditionally stable.

Predictor Corrector methods can also be used to increase the stability properties of the scheme.
