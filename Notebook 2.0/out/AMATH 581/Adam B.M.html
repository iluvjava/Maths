Integration

IVP given as

y′=f(t,y)y' = f(t, y) y′=f(t,y)

Where yyy could be a vector

Consider solving the problem with intergrations like:

y′(s)=ddt∫t0sf(t,y(t))dty'(s) = \frac{d}{dt}\int_{t_0}^s f(t, y(t))dt y′(s)=dtd​∫t0​s​f(t,y(t))dt

Which is saying that:

y(s)=∫t0sf(t,y(t))dty(s) = \int_{t_0}^s f(t, y(t))dt y(s)=∫t0​s​f(t,y(t))dt

Since we are interested in next time-step of the ODE, so what we are essentially doing is:

ytn+1=∫tntn+1f(t,y(t))dty_{t_n + 1} = \int_{t_n}^{t_{n+ 1}} f(t, y(t))dt ytn​+1​=∫tn​tn+1​​f(t,y(t))dt

And if this can be solved analytically, then we are good, however this is not always the case.

But the key here is to use approximation for the integral, with a polynomial and some finite differences, which will be useful later. Currently it looks not much different from the Taylor series's method for analyzing things.

f(t,y(t))≈f(tn,yn)f(t, y(t)) \approx f(t_n, y_n) f(t,y(t))≈f(tn​,yn​)

Instead of using the y(t)y(t)y(t), let's just use the yny_nyn​ from the previous time step as our point of evaluation.

yn+1=yn+∫tntn+1f(tn,yn)dty_{n+1} = y_n + \int_{t_n}^{t_{n+1}} f(t_n, y_n) dt yn+1​=yn​+∫tn​tn+1​​f(tn​,yn​)dt

Adam's Method

In this section we use the following notations:

f_n = f(t_n, y(t_n))$$ $$\Delta t = t_{n + 1} - t_{n}

Consider the approximation of the f(tn,y(tn))f(t_n, y(t_n))f(tn​,y(tn​)) in term of the following polynomials:

fn+fn−fn−1Δt(t−tn)f_n + \frac{f_n - f_{n-1}}{\Delta t}(t - t_n) fn​+Δtfn​−fn−1​​(t−tn​)

Then the time stepping scheme involves solving the following:

yn+1=yn+∫tntn+1fn+fn−fn−1Δt(t−tn)dty_{n + 1} = y_n + \int_{t_n}^{t_{n + 1}} f_n + \frac{f_n - f_{n-1}}{\Delta t}(t - t_n) dtyn+1​=yn​+∫tn​tn+1​​fn​+Δtfn​−fn−1​​(t−tn​)dt

Which in the end will give us:

yn+1=yn+Δt2(3fn−fn−1)y_{n+1} = y_n + \frac{\Delta t}{2}(3f_n - f_{n - 1}) yn+1​=yn​+2Δt​(3fn​−fn−1​)

This is an explicit scheme named: "Adams Bashforth"

Now we develop the implicit time stepping scheme using this approximation:

fn+fn−fn+1Δt(t−tn)f_n + \frac{f_n - f_{n+1}}{\Delta t}(t - t_n) fn​+Δtfn​−fn+1​​(t−tn​)

Not much changed but this time we use the future time stepping to improve our scheme here.

and the equation we are trying to solve here will be:

yn+1=yn+∫tntn+1fn+fn−fn+1Δt(t−tn)dty_{n + 1} = y_n + \int_{t_n}^{t_{n + 1}} f_n + \frac{f_n - f_{n+1}}{\Delta t}(t - t_n) dtyn+1​=yn​+∫tn​tn+1​​fn​+Δtfn​−fn+1​​(t−tn​)dt

and after solving it we have the time stepping scheme:

yn+1=yn+Δt2(fn+1+fn)y_{n+1} = y_n + \frac{\Delta t}{2}(f_{n + 1} + f_n) yn+1​=yn​+2Δt​(fn+1​+fn​)

And notice that, the term fn+1f_{n + 1}fn+1​ is going to need yn+1y_{n + 1}yn+1​, which is the next time step we need for things.

In this case, we have an implicit method and this method in particular is called: "Adams Moulton"

Multi_Stepping and Predictors

We can combine both of the implicit and the explicit time steppingg scheme together to get best of the both worlds.

Assume that we have previous 2 points, yn−1y_{n - 1}yn−1​ and yny _{n}yn​, then we can use the explicit method to get the future time step as:

yn+1p=yn+Δt2(3fn−fn−1)y^p_{n + 1} = y_n + \frac{\Delta t}{2}(3f_n - f_{n - 1}) yn+1p​=yn​+2Δt​(3fn​−fn−1​)

Now with the predict future time step, which might be unstable for certain functions, we are going to use this forthe implicit scheme to re-evaluate the value for yn+1y_{n + 1}yn+1​, and this is will be given as:

yn+1=yn+Δt2(f(tn+1,yn+1p)+fn)y_{n + 1} = y_n + \frac{\Delta t}{2} (f(t_{n + 1}, y^p_{n + 1}) + f_n) yn+1​=yn​+2Δt​(f(tn+1​,yn+1p​)+fn​)

And then this new yn+1y_{n + 1}yn+1​ is gotten from the predicted: yn+1ny_{n + 1}^nyn+1n​ from the explicit method. This is going to be a bit faster comparing just directly using the explicit method for the problem.

This Multi-Stepping scheme has an accuracy that varies a lot, and the ode311 is one of the multi-stepping method that uses this principle, and when the ODEs gets challenging, it's going to make the Δt\Delta tΔt smaller.

Predictor Corrector

Any explicit scheme combined with an explicit scheme's predictor is a valid predictor corrector method.

And hence we also have the option to combine the forward and the backward Euler together to get a predictor corrector method.

This is the Backwards Euler:

yn=yn−1+Δtf(tn,y~n)y_n = y_{n - 1} + \Delta tf(t_{n}, \tilde{y}_n) yn​=yn−1​+Δtf(tn​,y~​n​)

And then we can substitute the y~n\tilde{y}_ny~​n​ with a forward Euler:

y~n=yn−1+Δtf(tn−1,yn−1)\tilde{y}_n = y_{n - 1} + \Delta tf(t_{n - 1}, y_{n - 1}) y~​n​=yn−1​+Δtf(tn−1​,yn−1​)

So it will look like this:

yn=yn−1+Δtf(tn,yn−1+Δtf(tn−1,yn−1))y_n = y_{n - 1} + \Delta tf(t_{n}, y_{n - 1} + \Delta tf(t_{n - 1}, y_{n - 1})) yn​=yn−1​+Δtf(tn​,yn−1​+Δtf(tn−1​,yn−1​))

Then the average of the predictor and the solution from the implicit method will be used for the new time step.

The analysis goes hard after this point, and it's better to use a concrete functions to analyze how well this method works for certain equations.
