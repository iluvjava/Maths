The Damped Harmonic Oscillator

Here, we are going to look at the Damped Harmonic Oscillator again and use the idea of multiple time scale. The harmonic oscillator is set to be:

x′′+2ϵx′+x=0x'' + 2\epsilon x' + x = 0 x′′+2ϵx′+x=0

With the assumption on the initial condition that: x(0,0)=0x(0, 0) = 0x(0,0)=0, the perturbated system can be summarized to be:

x0(0,0)=x1(0,0)=x2(0,0)⋯=0x_0(0, 0) = x_1(0, 0) = x_2(0, 0) \cdots = 0 x0​(0,0)=x1​(0,0)=x2​(0,0)⋯=0

However, the initial condition that x′(0)=0x'(0) = 0x′(0)=0 can be summarized as:

(∂τ+ϵ∂T)(x0+ϵx1+ϵ2x2⋯ )=1(\partial_\tau + \epsilon \partial_T)(x_0 + \epsilon x_1 + \epsilon^2 x_2 \cdots ) = 1 (∂τ​+ϵ∂T​)(x0​+ϵx1​+ϵ2x2​⋯)=1

And this is like polynomials and we need to collect all the like terms using ϵ\epsilonϵ.

∂x0∂τ=1∂Tx0+∂Tx1=0∂Tx1+∂Tx2=0⋯\frac{\partial x_0}{\partial \tau} = 1 \quad \partial_Tx_0 + \partial_Tx_1 = 0 \quad \partial_T x_1 + \partial_T x_2 = 0 \quad \cdots ∂τ∂x0​​=1∂T​x0​+∂T​x1​=0∂T​x1​+∂T​x2​=0⋯

Now we have the initial conditions, let's take a look at the perturbed system:

(∂τ2+2ϵ∂τ∂T+ϵ2∂T2)(x0+ϵx1)+2ϵ(∂τ+ϵ∂t)(x0+ϵx1)+x0+ϵx1=0(1)(\partial_\tau^2 + 2\epsilon\partial_\tau\partial_T + \epsilon^2\partial_T^2)(x_0 + \epsilon x_1) + 2\epsilon (\partial_\tau + \epsilon \partial_t)(x_0 + \epsilon x_1) + x_0 + \epsilon x_1 = 0 \tag{1} (∂τ2​+2ϵ∂τ​∂T​+ϵ2∂T2​)(x0​+ϵx1​)+2ϵ(∂τ​+ϵ∂t​)(x0​+ϵx1​)+x0​+ϵx1​=0(1)

Notice that, for the expansion of perturbed solution, we only kept all the way up to first order.

Expanding, we an have the perturbed system represented as:

∂τx0+2ϵ∂τ∂Tx0+ϵ∂τ2x1+x0+ϵx1+2ϵ∂τx0+O(ϵ2)=0(2)\partial_\tau x_0 + 2\epsilon \partial_\tau\partial_Tx_0 + \epsilon \partial_\tau^2x_1 + x_0 + \epsilon x_1 + 2\epsilon \partial_\tau x_0 + \mathcal{O}(\epsilon^2) = 0 \tag{2} ∂τ​x0​+2ϵ∂τ​∂T​x0​+ϵ∂τ2​x1​+x0​+ϵx1​+2ϵ∂τ​x0​+O(ϵ2)=0(2)

And, some of the term of expansion on (1) has been pushed into O\mathcal{O}O, because we are only looking at the the first order perturbation expression for the perturbed system.

Zero Order Solution

Starting 0 th order perturbed system, we have system:

∂τ2x0+x0=0\partial_\tau^2 x_0 + x_0 = 0 ∂τ2​x0​+x0​=0

Homogenous second order linear system, and notice that:

x0=A(T)cos⁡(τ)+B(T)sin⁡(τ)x0(0,0)=0(3)x_0 = A(T)\cos(\tau) + B(T)\sin(\tau) \quad x_0(0, 0) = 0 \tag{3} x0​=A(T)cos(τ)+B(T)sin(τ)x0​(0,0)=0(3)

Notice that the integration constant is wrt the long time scale. Assuming that, it's somewhat changing wrt to the long time scale factor TTT.

Cancelling Resonance and find Integration Constants

Using (1), the perturbed first order system, include everything up to ϵ\epsilonϵ gives system:

∂τ2x1+x1=−2∂T∂τx0−2∂τx0(4)\partial_\tau^2x_1 + x_1 = -2\partial_T\partial_\tau x_0 - 2 \partial_\tau x_0 \tag{4} ∂τ2​x1​+x1​=−2∂T​∂τ​x0​−2∂τ​x0​(4)

Observe: LHS are first order perturbed, RHS are zero first perturbed.

Substitute (3) into (4) gives:

∂τ2x1+x1=2(A+A′)sin⁡(τ)+2(B+B′cos⁡(τ)\partial_\tau^2x_1 + x_1 = 2(A + A')\sin(\tau) + 2(B + B' \cos(\tau) ∂τ2​x1​+x1​=2(A+A′)sin(τ)+2(B+B′cos(τ)

Notice: ∂TA=A′\partial_T A = A'∂T​A=A′ and ∂TB=B′\partial_T B = B'∂T​B=B′

Assume: No resonance exists on time scale ttt, then, RHS in expression (4) is zero, means:

A′+A=0B′+B=0A' + A = 0 \quad B' + B = 0 A′+A=0B′+B=0

Means:

A=A0exp⁡(−T),B=B0exp⁡(−T)A = A_0 \exp(-T), B = B_0\exp(-T) A=A0​exp(−T),B=B0​exp(−T)

Now, we only have constant to determine, using initial condition:

x0(0,0)=0∂τx0(0,0)=1x_0(0, 0) = 0 \quad \partial_\tau x_0(0, 0) = 1 x0​(0,0)=0∂τ​x0​(0,0)=1

Get: A0=0,B0=1A_0 = 0, B_0 = 1A0​=0,B0​=1

Substitute back into (3) get:

x=exp⁡(−ϵt)sin⁡(t)+O(ϵ)x = \exp(-\epsilon t)\sin(t) + \mathcal{O}(\epsilon) x=exp(−ϵt)sin(t)+O(ϵ)

And this, is the first order perturbation approximation for the damped Harmonic Oscillator.

Summarization

  1. Use multiple time scale operator to transform the system with perturbation basis function.
  2. Solve the system wrt to different degree of ϵ\epsilonϵ.
    1. Solve for x0x_0x0​ on zeroth degree.
    2. Solve for x1x_1x1​ on first degree.
  3. Separate out the Harmonics on the Trigs, and set the x1x_1x1​ RHS to be zero, reducing the resonance.
  4. Solve for undetermined coefficients for the x0x_0x0​ system.
