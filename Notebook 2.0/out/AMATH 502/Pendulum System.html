The System

Linear pendulum system is just the non-linear one with the angle very close to zero. And this is the non-linear pendulum system:

θ′′+gLsin⁡θ=0θ∈[−π,π)\theta'' + \frac{g}{L}\sin\theta = 0 \quad \theta \in [-\pi, \pi) θ′′+Lg​sinθ=0θ∈[−π,π)

And we can view the system periodically, allowing us to take all the real values for the variable θ\thetaθ

Putting the 2nd order system into a 2D system, we will have:

[θv]′=[v−gLsin⁡θ]\begin{bmatrix} \theta \\ v \end{bmatrix}' = \begin{bmatrix} v \\ \frac{-g}{L}\sin\theta \end{bmatrix} [θv​]′=[vL−g​sinθ​]

And substitution is θ′=v\theta' = vθ′=v.

Non-Linear System is Conservative (Not Damped)

The system is conservative if we can find the potential function. Take note that:

v′=gLsin⁡θ=ddθ(−gLcos⁡θ)v' = \frac{g}{L}\sin\theta = \frac{d}{d\theta} \left( -\frac{g}{L}\cos\theta \right) v′=Lg​sinθ=dθd​(−Lg​cosθ)

And hence, we have a way of representing the forces wrt to the angle, which is:

θ′′=ddθ(−gLcos⁡θ)\theta'' = \frac{d}{d\theta} \left( -\frac{g}{L}\cos\theta \right) θ′′=dθd​(−Lg​cosθ)

So then the potential function wrt to θ\thetaθ is looking like:

V(θ)=gLcos⁡(θ)V(\theta) = \frac{g}{L}\cos(\theta) V(θ)=Lg​cos(θ)

Another way to find it is integrating with an integrator, giving me:

θ′′+gLsin⁡θ=0θ′θ′′+θ′gLsin⁡θ=0ddt((θ′)22+gLcos⁡θ)=0(θ′)22+gLcos⁡θ=E\begin{align} \theta'' + \frac{g}{L}\sin\theta &= 0 \\ \theta'\theta'' + \theta'\frac{g}{L}\sin\theta &= 0 \\ \frac{d}{dt}\left( \frac{(\theta')^2}{2} + \frac{g}{L}\cos\theta \right) &= 0 \\ \frac{(\theta')^2}{2} + \frac{g}{L}\cos\theta &= E \end{align} θ′′+Lg​sinθθ′θ′′+θ′Lg​sinθdtd​(2(θ′)2​+Lg​cosθ)2(θ′)2​+Lg​cosθ​=0=0=0=E​​

And in this case, we also identified the energy function, which is the LHS of the equation.

Fixed Points of the System (Not damped)

Fixed point happens when V′(θ)=0V'(\theta) = 0V′(θ)=0, which means that sin⁡(θ)=0\sin(\theta) = 0sin(θ)=0, which means that: θ=0,±π\theta = 0, \pm\piθ=0,±π, and periodically, it's really the same point for ±π\pm \pi±π.

In addition for V′(θ)=0V'(\theta) = 0V′(θ)=0, we also need to make the pendulum stationary, meaning that: v=0v = 0v=0.

Therefore the fixed points of the system will be:

p1=[00]p2=[±π0]p_1 = \begin{bmatrix} 0 \\ 0 \end{bmatrix}\quad p_2 = \begin{bmatrix} \pm \pi \\ 0 \end{bmatrix} p1​=[00​]p2​=[±π0​]

And then the Jacobian is going to be:

J=[01−gLcos⁡θθ]J = \begin{bmatrix} 0 & 1 \\ \frac{-g}{L}\cos\theta & \theta \end{bmatrix} J=[0L−g​cosθ​1θ​]

Substituting the fixed points we will have the matrices:

J(p1)=[01−gL0]J(p2)=[01gL0]J(p_1) = \begin{bmatrix} 0 & 1 \\ \frac{-g}{L} & 0 \end{bmatrix} \quad J(p_2) = \begin{bmatrix} 0 & 1 \\ \frac{g}{L} & 0 \end{bmatrix} J(p1​)=[0L−g​​10​]J(p2​)=[0Lg​​10​]

Notice that for J(p1)J(p_1)J(p1​), the eigenvalues are: ±ig/L\pm i\sqrt{g/L}±ig/L​ meaning that it is a stationary fixed point. However, as summarized in: [[2D Non-Linear System]]. In the case of Linear analysis giving us the result of a neutrally stable center, we should not trust the result.

Question: Should we trust the result of linear analysis?

Answer: Yes, because we showed that the system is conservative, and when it's conservative, all centers will have to be Neutrally stable!

Let's consider the other fixed point: J(p2)J(p_2)J(p2​) which gives us the eigenvalues: ±g/L\pm \sqrt{g/L}±g/L​, and it has corresponding eigenvectors:

v1=[L/g1]v2=[−L/g1]v_1 = \begin{bmatrix} \sqrt{L/g} \\ 1 \end{bmatrix}\quad v_2 = \begin{bmatrix} -\sqrt{L/g}& \\1 \end{bmatrix} v1​=[L/g​1​]v2​=[−L/g​1​]

And so this is a saddle node, with v1v_1v1​ being the unstable manifold, and v2v_2v2​ being the stable manifold.
