Intro:

To start with, we need to look at 2D autonomous Linear system.

\begin{align} x_1' &= ax_1 + bx_2\\ x_2' &= cx_1 + dx_2 \end{align}

Where all the constants are real number and this is a system that can be expressed with matrices like:

x′=Axx' = Ax x′=Ax

Nice and easy.

2D Linear System

2 Facts for Linear system:

  1. Principals of Super Position: if zzz and yyy both solve the linear system, then the linear combinations of these 2 solutions are going to solve the system as well.
  2. x=0x = \mathbf{0}x=0 is always a fixed point. This is obvious cause this is a linear system.

The system is like:

mx′′+hx=0mx'' + hx = 0 mx′′+hx=0

Stability, Attractiveness of Fixed Points

These definitions are more general, and it can be applied to stuff that is not linear.

  • Lyapounov Stability: A fixed point x∗x^*x∗ is Lyapounov Stable if all trajectory starts near it, stays there.

    For all x(t)x(t)x(t) with x0x_0x0​ close to x∗x^*x∗ as t→∞t \rightarrow \inftyt→∞, ∣x(t)−x∗∣|x(t) - x^*|∣x(t)−x∗∣ is bounded.

    Like planetary motions.

  • Attractive fixed point: Starts near it will approach it as t→∞t\rightarrow \inftyt→∞.

    For all x(t)x(t)x(t), t>0t > 0t>0 with x0x_0x0​ close to x∗x^*x∗, lim⁡t→∞x(t)=x∗\lim_{t\rightarrow\infty} x(t) = x^*limt→∞​x(t)=x∗

  • Asymptoptically Stable: Stable and attractive.

Note: Attractive doesn't mean stable, consider multiple points that are stable, or the case where it's periodic. This also means that, a set of fixed points (or a fixed region) can also be fit into the above definition.

Eigenvalue Classification of Linear System.

Two Real Positive Eigenvalues

λ1>λ2>0\lambda_1 > \lambda_2 > 0λ1​>λ2​>0

The absolute value of the solution is going to infinite, All the trajectory are going way from origin, all the way to infinity.

x(t)=c1exp⁡(λ1t)+c2exp⁡(λ2t)x(t) = c_1 \exp(\lambda_1 t) + c_2 \exp(\lambda_2 t) x(t)=c1​exp(λ1​t)+c2​exp(λ2​t)

Asymptotically Speaking, it's going in the λ1\lambda_1λ1​ eigenvector's direction, asymptotically.

That means the origin is **Unstable **, which is called an Unstable Node.

Two Real Eigenvalues That are Both Negative

λ1,λ2<0\lambda_1, \lambda_2 < 0λ1​,λ2​<0

It's very similar to the previous case, it's heading towards the origin. All the trajectories are in reverse compare to the previous case.

Now the origin is Stable, and this is called a Stable Node.

Two Distinct Read Eigenvalues, Opposite Signs.

λ1λ2>0\lambda_1\lambda_2 > 0λ1​λ2​>0. It's approaching the fixed points in the eigenvector corresponding to the negative eigenvalues, and it's going away from the fixed point in the direction of the eigenvector of the positive eigenvalue.

This is called a Saddle Point, and it's Unstable
