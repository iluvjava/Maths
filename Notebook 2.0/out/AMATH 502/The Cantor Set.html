The Similarity Dimension

This only works for self similar sets.

Suppose that, a self-similar set consists of mmm copies of itself scaled down by a factor of RRR.

How many copies of itself it has when we scale it down (zoom in)?

Then the dimension is:

d=ln⁡(M)ln⁡(R)M=Rdd = \frac{\ln(M)}{\ln(R)} \quad M = R^d d=ln(R)ln(M)​M=Rd

Example: The Line Segment

A line segment is a line segment when we zoom in, this is self similarity.

If we cut it in half we get two copies of itself, scaled down by a factor of 2.

This means that, M=2M = 2M=2, and R=2R = 2R=2, we have 2 copies when we scaled zoom in by 2x.

Then we have:

ln⁡(2)ln⁡(2)=1=d\frac{\ln(2)}{\ln(2)} = 1 = d ln(2)ln(2)​=1=d

The line segment has a dimension of 1. Which is what we want really.

Example: Consider a Square

We can cut it into 4 little square, evenly, then we have M=4M = 4M=4, we will need to zoom in by 2x to get the sub-square looking like the super-square.

This means that M=4M = 4M=4, while R=2R = 2R=2, which means that:

ln⁡(4)ln⁡(2)=d=2\frac{\ln(4)}{\ln(2)} = d= 2 ln(2)ln(4)​=d=2

Example: Consider Cantor Set

We made 2 copies of the same set, and each of the copies are 1/31/31/3 in size of the original, so we need to zoom in by 3x to get the same shape.

This means that: M=2M = 2M=2 and R=3R = 3R=3.

And this means that

d=ln⁡2ln⁡3=0.63...d = \frac{\ln 2}{\ln 3} = 0.63... d=ln3ln2​=0.63...

This is a set with dimension that is between zero and one. Which is making sense because it has Lebsegue Measure of zero.
