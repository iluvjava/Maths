Intro

This file seeks to explain and explore further about the idea of convex conjugate. The concept first appears here:

Fenchel Conjugate

It's also called the Fenchel Legendre Transform.

Let f(x):Rn↦R∪{−∞,∞}f(x): \mathbb{R}^n \mapsto \mathbb{R} \cup \{-\infty, \infty\}f(x):Rn↦R∪{−∞,∞}, then the Fenchel Conjugate is defined to be:

f∗(y)=sup⁡x∈E{yTx−f(x)}f^*(y) = \sup_{x \in E} \left\lbrace y^Tx - f(x) \right\rbrace f∗(y)=x∈Esup​{yTx−f(x)}

Conjugate Epigraph contains all Affine Minorants of the Original Function

(y,z)∈epir(f)(y, z) \in \underset{r}{\text{epi}}(f) (y,z)∈repi​(f)

r≥z≥f∗(y)≥yTx−f(x)  ⟹  f(x)≥yTx−z∀xr \ge z \ge f^*(y) \ge y^Tx - f(x) \implies f(x) \ge y^Tx - z \quad \forall x r≥z≥f∗(y)≥yTx−f(x)⟹f(x)≥yTx−z∀x

Fenchel's Conjugate Properties

Double Conjugate making it Smaller.

f∗∗(x)≤f(x)f^{**}(x) \le f(x) f∗∗(x)≤f(x)

Such a claim make sense geometrically, a convex envelop is reconstructed using the conjugate of the original function, and it will always be smaller in quantity.

Proof

f∗∗(z)=sup⁡x{zTx−f∗(x)}=sup⁡x{zTx−sup⁡y{zTy−f(y)}}  ⟹  [1]sup⁡x{inf⁡y{zT(x−y)+f(y)}}≤inf⁡y{sup⁡x{zT(x−y)+f(y)}}\begin{aligned} f^{**}(z) = \sup_x \left\lbrace z^Tx - f^*(x) \right\rbrace & = \sup_x \left\lbrace z^Tx - \sup_{y} \left\lbrace z^Ty - f(y) \right\rbrace \right\rbrace \\\underset{[1]}{\implies} \sup_x \left\lbrace \inf_{y} \left\lbrace z^T(x - y) + f(y) \right\rbrace \right\rbrace &\le \inf_y \left\lbrace \sup_x \left\lbrace z^T(x - y) + f(y) \right\rbrace \right\rbrace \end{aligned} f∗∗(z)=xsup​{zTx−f∗(x)}[1]⟹​xsup​{yinf​{zT(x−y)+f(y)}}​=xsup​{zTx−ysup​{zTy−f(y)}}≤yinf​{xsup​{zT(x−y)+f(y)}}​

[1]: Using Minmax, Maxmin Lemma, introduced in [[MinMax MaxMin Lemma]]

Note:

Equality is true when the function is convex, because a convex set can be reconstructed by all the half spaces (The Affine Minorants).

Fenchel's Identity

Fenchel's Identity is useful because it links the dual and primal variable, and can be exploited for dualization of problems.

Claim 1

The dual variable is in the sub gradient of the original function. (Make sense geometrically), The argument is reversible, which means that:

z∈∂f(x)(1)z \in \partial f(x) \tag{1} z∈∂f(x)(1)
