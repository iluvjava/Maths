Euclidean Distance is one of the Bregman Divergences.

Bregman Divergence

It's parameterized by a function FFF, and it takes 2 points and measures their divergence with function FFF. Lets denote Df(x,y)D_f(x, y)Df​(x,y) be the Bregman divergence for 2 points x,yx, yx,y. Here is a list of things about it:

  • Non-negative if FFF is convex.
  • Convex in the first argument ppp.
  • Linearity

DF1+λF2(p,q)=DF1(p,q)+λDF2(p,q)D_{F1 + \lambda F_2}(p, q) = D_{F1}(p, q) + \lambda D_{F2}(p, q) DF1+λF2​​(p,q)=DF1​(p,q)+λDF2​(p,q)

  • Duality, the function FFF is has a convex conjugate function, then:

DF⋆(∇F(p),∇F(q))=DF(p,q)D_{F\star}(\nabla F(p), \nabla F(q)) = D_F(p, q) DF⋆​(∇F(p),∇F(q))=DF​(p,q)

Bregman Divergence definition:

Dφ(x,z)=φ(x)−φ(z)−⟨∇φ(z),x−z⟩D_\varphi (x, z) = \varphi(x) - \varphi(z) - \langle \nabla \varphi(z), x - z \rangle Dφ​(x,z)=φ(x)−φ(z)−⟨∇φ(z),x−z⟩

Anchoring it at the point zzz the distance between the actual value of the function and the value predicted by the gradient will be the Bregman Divergence.

Warn:

Gregman Divergence is not symmetric.
