Separation of Variables: Setting things up (Dirichlet Boundary Conditions, Heat Equation)

Let's say that the solution is the product of 2 functions, then it will be like:

u(x,t)=X(x)T(t)u(x, t) = X(x)T(t) u(x,t)=X(x)T(t)

It's like, there is a function that evolves with along the time dimension, and then scaled by another function, as the time variable changes.

The PDE is:

ut=c2uxx0<x<Lt>0u_t = c^2u_{xx} \quad 0 < x < L \quad t> 0 ut​=c2uxx​0<x<Lt>0

The initial condition is:

u(x,0)=F(x)u(x, 0) = F(x) u(x,0)=F(x)

Where we keep the generality by assuming the initial condition distribution function F(x)F(x)F(x).

And the boundary condition is:

u(0,t)=u(L,t)=0u(0, t) = u(L, t) = 0 u(0,t)=u(L,t)=0

For the separation variable to work, the boundary will have to be zero. What if we are using fahrenheit? The method of separation of variable won't work for this boundary conditions. If, the boundary condition is fixed with the same value, then we can make a substitution, say v=u−32v = u - 32v=u−32, so the boundary condition is fixed at zero.

Let's summarize the procedure:

  1. Assume that uuu is the product of 2 functions like: X(x)T(x)X(x)T(x)X(x)T(x), for this PDE, it works well.
  2. Substitute the separation form into the equation.
  3. After separation, set everything to an undetermined constant. Try different cases for the constant.
