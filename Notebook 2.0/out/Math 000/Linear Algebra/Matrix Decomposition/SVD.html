Singular Value Decomposition

Notations: * is the complex conjugate transpose

Relevant Resources: link1 Mit Lecture Notes on SVD: link2,

  • Singular eigen vector of AAA and ATAA^TAATA
  • Applied to matrix of all sizes, real and complex, M=UΣVM = U\Sigma VM=UΣV sometimes it's written differently as M=UΣV∗M = U\Sigma V^*M=UΣV∗, but it's the same thing and it's one conjugate transpose away.
    • U,VU, VU,V are ortho matrices. Σ\SigmaΣ is a diagonal matrix, don't have to be squared, and that is how we can get full SVD.
    • M∈Rm×nM\in \mathbb{R}^{m \times n}M∈Rm×n
    • U∈Rm×mU\in \mathbb{R}^{m \times m}U∈Rm×m
    • Σ∈Rm×n\Sigma \in \mathbb{R}^{m \times n}Σ∈Rm×n
    • V∈Rn×nV\in \mathbb{R}^{n \times n}V∈Rn×n
  • A bit of a intuitions
    • It's like eigen decomposition, but we have 2 ortho matrices that "adapts" vector from intput subspace and output subspace.

Ideas Behind

  • Facts about Symmetric Matrices (Or Hermitian):
    • Has Only real Eigen values
    • Always Diagonalizable
    • Has orthogonal Eigenvectors
  • Then, using this property, we can say the following about the eigen decomposition of MMM and the SVD decomposition of M.
  • Consider M∗MM^*MM∗M
    • M∗M=(UΣV)∗(UΣV)=(V∗Σ∗U∗)(UΣV)M^*M =(U\Sigma V)^*(U\Sigma V) = (V^*\Sigma^* U^*)(U\Sigma V)M∗M=(UΣV)∗(UΣV)=(V∗Σ∗U∗)(UΣV)
    • M∗M=V∗(Σ∗Σ)VM^*M =V^*(\Sigma^*\Sigma) VM∗M=V∗(Σ∗Σ)V
  • Similarlly, we can Harmitant the other way and get:
    • MM∗=U(ΣΣ∗)U∗MM^* = U(\Sigma\Sigma^*)U^*MM∗=U(ΣΣ∗)U∗
  • Now, we use the properties of Symmetric/Harmitian Matrices:
    • V∗V^*V∗, as a Ortho Matrix, is the Eiven space of M∗MM^*MM∗M and similarly, UUU is the Ortho Matrix, of Eigen Space of MM∗  MM^*\;MM∗
    • Σ=D\Sigma = \sqrt{D}Σ=D​ where D is the Diagonal Matrix of the Eigen values of M∗MM^*MM∗M or MM∗MM^*MM∗ (Depending on whether it's Skinnier or Fat...?) Also, agumentation padded with zeroes are needed to get non-squared Σ\SigmaΣ matrix.
    • More on Symmetric/Hermitian Matrices: [[Hermitian Adjoint]]
  • Geometric Interpretations:
    • The matrix does nothing but the following: It rotates and stretches the inputs vector and that is all what it does, and hence, the unitary matrix UUU and the unitary matrix VVV.
    • The transformation is literally, decompose any vector into an ortho basis, rotates the bases and the stretch the vector along the directions of the basis vector, and then deform the hypershphere from the input space into a eclipse in the hyper space.
  • Subspace Interpretations
    • V∗V^*V∗ Describes the inputs in alternative ortho subspace under the input space Rn\mathbb{R}^nRn.
    • Σ\SigmaΣ Takes in the description from V∗V^*V∗ and then scaled it a bit.
    • UUU Rotates, change orietation, on the output space Rm\mathbb{R}^mRm
  • Something Interesting to Notice:
    • Observe that, one the third step of the derivation, it is implied that A∗AA^*AA∗A and AA∗AA^*AA∗ are sharing the same set of eigen values (ginore zeroes), if such a decomposition exists, this will be a good way to check things which I don't know why yet but its interesting.

How To do SVD

  • Find the eigen vec and values for ATAA^TAATA
  • Order the Eigen values in decreasing order for Σ\SigmaΣ, and remember to take the square root of those eigen values of ATAA^TAATA
  • Get UUU matrix using AV=UΣAV = U\SigmaAV=UΣ

Full SVD

  • Just pad Σ\SigmaΣ with zeroes so that the matrix UUU can be a square matrix.

Things to Take Notice

It's not that easy we still need to take care of somethings when doing SVD.

Ordering of the Eigen Values

  • The values on the entry for the Σ\SigmaΣ matrix is in descending order, re-ordering of rows in V∗V^*V∗ and the columns in UUU won't change the transformation, because the V∗V^*V∗ will be able to find the correct representation on the ortho basis. More specifically:
    • (Σ)i,i(\Sigma)_{i, i}(Σ)i,i​ is the iii th eigen values, in desceding order, of the matrix ATAA^TAATA
    • (V):,i(V)_{:,i}(V):,i​ is the eigen vector for the corresponding eigen value (Σ)i,i(\Sigma)_{i, i}(Σ)i,i​ for ATAA^TAATA
