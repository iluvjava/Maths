There are a lot of theorem relevant to the Singular Value decomposition and they are really important to understanding on numerical Linear Algebra.

Here we introduce theorems that are needed to show the the SVD decomposition for a matrix indeed, does work the way we want it to be.

Theorems on Singular Values

SVD Theorem 1:

The nonzero singular values of A are the square roots of the nonzero eigenvalue of AAHAA^HAAH or AHAA^HAAHA

Proof sketch:

  1. The matrix AHAA^HAAHA and AAHAA^HAAH share the same nonzero eigen values.
  2. Using the (1.), we know that the eigen decomposition of the the Hermitian matrix will have the diagonal matrix with the same non zero scalars. (By properties of the Hermitian Adjoin Operator)
  3. Link the singular values with the Eigen Decomposition of the AHAA^HAAHA and the AAHAA^HAAH

Proof on (1.0):

Suppose that λ≠0\lambda\neq 0λ=0 is an eigen value for the matrix AHAA^HAAHA, say that xxx is the corresponding eigen vector for that eigen value. Then we can say the following:

AHAx=λxA^HAx = \lambda x AHAx=λx

Now consider:

AAH(Ax)=A(λx)=λAxAA^H(Ax) = A(\lambda x) = \lambda Ax AAH(Ax)=A(λx)=λAx

And then we will know that, the value λ\lambdaλ is also an eigen value for the matrix AAHAA^HAAH and the corresponding eigen vector is AxAxAx

Notice that, we can also start with AAHy=γyAA^Hy = \gamma yAAHy=γy, then we can prove in the other direction and find the equivalence.

Therefore, The matrix AHAA^HAAHA and the matrix AAHAA^HAAH has the same set of non-zero eigen values.

SVD Theorem 2:

A=AH  ⟹  A = A^H \impliesA=AH⟹ the singular values are the absolute values of the eigen values of A.

Proof:

The the eigen decomposition of matrix A be represented as A=QΛQHA = Q \Lambda Q^HA=QΛQH where Λ\LambdaΛ is a diagonal matrix of real numbers. (by properties of Hermitian matrix and the matrix AAA is Hermitian)

Consider the product AHAA^HAAHA if the eigen decomposition is given:

AHA=(QΛQH)H(QΛQH)=(QΛQH)(QΛQH)=QΛ2QHA^HA = (Q\Lambda Q^H)^H(Q\Lambda Q^H) = (Q\Lambda Q^H)(Q\Lambda Q^H) = Q\Lambda^2 Q^H AHA=(QΛQH)H(QΛQH)=(QΛQH)(QΛQH)=QΛ2QH

On the other hand we also know that the Eigen decomposition of the matrix AHAA^HAAHA is linked to the singular values of the matrix AAA by the formuta(By SVD of matrices):

AHA=VΣ2VHA^HA = V\Sigma^2 V^H AHA=VΣ2VH

And both expression are the Eigen decomposition of the matrix AHAA^HAAHA, and here we will be able to obtain the following relationship:

Λ2=Σ=∣Λ∣\sqrt{\Lambda^2} = \Sigma = |\Lambda| Λ2​=Σ=∣Λ∣

And hence, the statement "The Singular Value are the absolute values of the eigen values of A" is proven.
