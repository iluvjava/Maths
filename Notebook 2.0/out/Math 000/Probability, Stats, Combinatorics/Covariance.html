Intro

Let X,YX, YX,Y be 2 random variables, then we define the covariance to be:

Cov(X,Y)=E[(X−E[X])(Y−E[Y])]=E[XY]−E[X]E[Y]\text{Cov}(X, Y) = \mathbb{E}\left[ (X -\mathbb{E}\left[X\right]) (Y -\mathbb{E}\left[Y\right]) \right] = \mathbb{E}\left[XY\right] - \mathbb{E}\left[X\right] \mathbb{E}\left[Y\right] Cov(X,Y)=E[(X−E[X])(Y−E[Y])]=E[XY]−E[X]E[Y]

Which is somewhat similar to variance, especially the second way of viewing it.

The application of covariance are striaght forward.

Covariance is a measurement of linear dependence between 2 random variable, it's testing some underlying generate process such as the Y=aX+bY = aX + bY=aX+b. Where one of the random variable is linear related to the other one.

Useful Properties

  1. X⊥Y  ⟹  Cov(X,Y)=0X\perp Y\implies \text{Cov}\left(X,Y\right) = 0X⊥Y⟹Cov(X,Y)=0, Where X⊥YX\perp YX⊥Y means that these 2 random variables are independent.
  2. Cov(X,X)=Var[X]\text{Cov}(X,X) = \text{Var}\left[X\right]Cov(X,X)=Var[X]
  3. Cov(X,Y)=Cov(Y,X)\text{Cov}\left(X,Y\right) = \text{Cov}\left(Y,X\right)Cov(X,Y)=Cov(Y,X)
  4. Cov(X+c,Y)=Cov(X,Y)\text{Cov}\left(X + c,Y\right) = \text{Cov}\left(X,Y\right)Cov(X+c,Y)=Cov(X,Y), shifting one of the variable won't change the covariance.
  5. Cov(aX+bY,Z)=aCov(X,Z)+bCov(Y,Z)\text{Cov}\left(aX + bY, Z\right) = a\text{Cov}\left(X,Z\right) + b \text{Cov}\left(Y,Z\right)Cov(aX+bY,Z)=aCov(X,Z)+bCov(Y,Z), distributive and constants can be taken out.
  6. Var[X+Y]=Var[X]+Var[Y]+2Cov(X,Y)\text{Var}\left[X + Y\right] = \text{Var}\left[X\right] + \text{Var}\left[Y\right] + 2 \text{Cov}\left(X,Y\right)Var[X+Y]=Var[X]+Var[Y]+2Cov(X,Y), think of the quadratic formula for (X+Y)2(X + Y)^2(X+Y)2 to remember this.
  7. Cov(∑i=1nXi,∑i=1nYi)=∑i=1n∑i=1mCov(Xi,Yj)\text{Cov}\left(\sum_{i = 1}^{n}X_i,\sum_{i = 1}^{n}Y_i\right) = \sum_{i = 1}^{n}\sum_{i = 1}^{m}\text{Cov}\left(X_i,Y_j\right)Cov(∑i=1n​Xi​,∑i=1n​Yi​)=∑i=1n​∑i=1m​Cov(Xi​,Yj​)

Property 5, 6, 7 are interesting, let's look into it.

Property 5

Cov(aX+bY,Z)=E[aXZ+bYZ]−E[aX+bY]E[Z]=aE[X]E[Z]+bE[Y]E[Z]−(aE[X]+bE[Y])E[Z]=aE[X]E[Z]+bE[Y]E[Z]−aE[X]E[Z]−bE[Y]E[Z]=aCov(X,Z)+bCov(Y,Z)\begin{aligned} \text{Cov}\left(aX + bY,Z\right) &= \mathbb{E}\left[aXZ + bYZ\right] - \mathbb{E}\left[aX + bY\right]\mathbb{E}\left[Z\right] \\ &= a \mathbb{E}\left[X\right]\mathbb{E}\left[Z\right] + b \mathbb{E}\left[Y\right]\mathbb{E}\left[Z\right] - ( a\mathbb{E}\left[X\right] + b \mathbb{E}\left[Y\right] )\mathbb{E}\left[Z\right] \\ &= a \mathbb{E}\left[X\right]\mathbb{E}\left[Z\right] + b \mathbb{E}\left[Y\right]\mathbb{E}\left[Z\right] - a \mathbb{E}\left[X\right]\mathbb{E}\left[Z\right] - b \mathbb{E}\left[Y\right]\mathbb{E}\left[Z\right] \\ &= a \text{Cov}\left(X,Z\right) + b \text{Cov}\left(Y,Z\right) \end{aligned} Cov(aX+bY,Z)​=E[aXZ+bYZ]−E[aX+bY]E[Z]=aE[X]E[Z]+bE[Y]E[Z]−(aE[X]+bE[Y])E[Z]=aE[X]E[Z]+bE[Y]E[Z]−aE[X]E[Z]−bE[Y]E[Z]=aCov(X,Z)+bCov(Y,Z)​

An alternative proof involves using property 5

Property 6

Var[X+Y]=E[(X+Y−E[X+Y])2]=E[(X−E[X]+Y−E[Y])2]=E[(X−E[X])2+(Y−E[Y])2+2(X−E[X])(Y−E[Y])]=Var[X]+Var[Y]+2Cov(X,Y)\begin{aligned} \text{Var}\left[X+ Y \right] &= \mathbb{E}\left[ (X + Y - \mathbb{E}\left[ X + Y \right])^2 \right] \\ &= \mathbb{E}\left[ (X - \mathbb{E}\left[X\right] + Y - \mathbb{E}\left[Y\right])^2 \right] \\ &= \mathbb{E}\left[ (X - \mathbb{E}\left[X\right])^2 + (Y - \mathbb{E}\left[Y\right])^2 + 2(X - \mathbb{E}\left[X\right])(Y - \mathbb{E}\left[Y\right]) \right] \\ &= \text{Var}\left[X\right] + \text{Var}\left[Y\right] + 2 \text{Cov}\left(X,Y\right) \end{aligned} Var[X+Y]​=E[(X+Y−E[X+Y])2]=E[(X−E[X]+Y−E[Y])2]=E[(X−E[X])2+(Y−E[Y])2+2(X−E[X])(Y−E[Y])]=Var[X]+Var[Y]+2Cov(X,Y)​

ρ(X,Y)=Cov(X,Y)Var[X]Var[Y]\rho(X, Y) = \frac{\text{Cov}\left(X,Y\right)} {\sqrt{\text{Var}\left[X\right]\text{Var}\left[Y\right]}} ρ(X,Y)=Var[X]Var[Y]​Cov(X,Y)​
