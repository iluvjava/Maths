The Definition of Divergence of a vector field

Vector field is a mapping such that: Fector field ---> Scalar field.

For a given function that Rn↦Rn\mathbb{R}^{n} \mapsto \mathbb{R}^{n}Rn↦Rn,

And the function is like:

F(x,y,z)=[F1(x,y,z)F2(x,y,z)F3(x,y,z)]F(x, y, z) = \begin{bmatrix} F_1(x, y, z)\\[0.5em] F_2(x, y, z)\\[0.5em] F_3(x, y, z) \end{bmatrix}F(x,y,z)=⎣⎢⎢⎡​F1​(x,y,z)F2​(x,y,z)F3​(x,y,z)​⎦⎥⎥⎤​

∇⋅(F)=∂F1∂x+∂F2∂y+∂F3∂x\nabla\cdot(F) = \frac{\partial F_1}{\partial x} + \frac{\partial F_2}{\partial y} + \frac{\partial F_3}{\partial x} ∇⋅(F)=∂x∂F1​​+∂y∂F2​​+∂x∂F3​​

Netflow in equals to netflow out means that ∇⋅F=0\nabla\cdot F = 0∇⋅F=0, and if this is applied fluid flow, then this is like incompressible fluids!
