A Fact

Assuming that the function as a Pole (Isolated Singularity) with order nnn at z0z_0z0​ (not Essential Singularity), then the following statement regarding the function and the Pole is true:

argminn(lim⁡z→z0(z−z0)nf(z)<∞)\underset{n}{\text{argmin}} \left(\lim_{z \rightarrow z_0} (z - z_0)^n f(z) < \infty \right) nargmin​(z→z0​lim​(z−z0​)nf(z)<∞)

Where n is the minimum positive integer needed such that the limit of the expression exists as a finite number.

There is a limit defined there, if I give the function a multiplier (z−z0)n(z - z0)^n(z−z0)n .

Comment

When evaluating the limit, it's really common to use the Lopital's Rule for the evaluation of the fraction, if it's in the undetermined form.

Trickier Example

Notice that, we cannot just use this as a way to eliminate the numerator that is approaching zero and then claim that we have the poles of some kind order, that is just terrible because consider the following example:

cos⁡(z)(z−π/2)4\frac{\cos(z)}{(z - \pi/2)^4} (z−π/2)4cos(z)​

And then consider the the point z=π/2z = \pi/2z=π/2. At first it will be really tempting to just claim that the point is a 4th order pole because multiplying it by (z−π/2)(z - \pi/2)(z−π/2) will give us a limit that evaluates to a constant, but please also consider that the limit:

lim⁡z→π/2cos⁡(z)(z−π/2)4\lim_{z \rightarrow \pi/2} \frac{\cos(z)}{(z - \pi/2)^4} z→π/2lim​(z−π/2)4cos(z)​
