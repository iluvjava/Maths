So, the trig functions and the hyperbolic Trig Functions can be extended to the complex numbers too. prereq: [[Analytic Functions (Exp, Poly)]]

Intro

The complex trig function is an analytic extension of the real trig functions, and in this case, it preserves all of the properties of the real Trigs function, and it's derived from the complex extension of the exponential function. However, it has interesting connection to the hyperbolic functions.

Def

cos⁡(z)=12(exp⁡(iz)+exp⁡(−iz))\cos(z) = \frac{1}{2}\left(\exp(iz) + \exp(-iz)\right) cos(z)=21​(exp(iz)+exp(−iz))

where the exp⁡\expexp function is the exponential function we derived earlier.

sin⁡(z)=12i(exp⁡(iz)−exp⁡(−iz))\sin(z) = \frac{1}{2i}\left(\exp(iz) - \exp(-iz)\right) sin(z)=2i1​(exp(iz)−exp(−iz))

Yeah, this is literally taking out the real and the complex part of the complex number, but this time, we do it on the polar form with the complex conjugate. Easy.

Properties Preserved:

  • Derivative still follows the same rules.
  • The symmetry where cos⁡(z)=cos⁡(−z)\cos(z) = \cos(-z)cos(z)=cos(−z) and sin⁡(z)=−sin⁡(z)\sin(z) = -\sin(z)sin(z)=−sin(z), so the function is still odd, and even for the sin⁡\sinsin and the cos⁡\coscos function.
  • Pythagorean Theorem: sin⁡(z)2+cos⁡(z)2=1\sin(z)^2 + \cos(z)^2 = 1sin(z)2+cos(z)2=1 is true.
  • tan⁡(z)=sin⁡(z)cos⁡(z)\tan(z) = \frac{\sin(z)}{\cos(z)}tan(z)=cos(z)sin(z)​, still true, and hence, all other trigs functions are going to be defined too after the sin⁡\sinsin and cos⁡\coscos.

New Properties:

  • cos⁡(z)=4\cos(z) = 4cos(z)=4 is something that has a solution to now, so that is kinda cool.

Hyperbolic

Let's investigate the connections between these 2 things. So it's like:

cosh⁡(x)=ez+e−z2\cosh(x) = \frac{e^z + e^{-z}}{2} cosh(x)=2ez+e−z​

sinh⁡(x)=ez−e−z2\sinh(x) = \frac{e^z - e^{-z}}{2} sinh(x)=2ez−e−z​

So then we can also express the trig function with the hyperbolic functions, and it's like:

cos⁡(iz)=cosh⁡(z)\cos(iz) = \cosh(z) cos(iz)=cosh(z)

sin⁡(iz)=isinh⁡(z)\sin(iz) = i\sinh(z) sin(iz)=isinh(z)

A More Scary Variation

cos⁡(z)=12(exp⁡(ix−y)+exp⁡(−ix+y))\cos(z) = \frac{1}{2}(\exp(ix - y) + \exp( -ix + y)) cos(z)=21​(exp(ix−y)+exp(−ix+y))

z=x+yiz = x + yiz=x+yi as a direct substitution.

12(e−y(cos⁡(x)+isin⁡(x))+ey(cos⁡(x)−isin⁡(x)))\frac{1}{2}(e^{-y}(\cos(x) + i\sin(x)) + e^{y}(\cos(x) - i\sin(x))) 21​(e−y(cos(x)+isin(x))+ey(cos(x)−isin(x)))

Collect the like terms:

12(cos⁡(x)(ey−e−y)+isin⁡(x)(e−y−ey))\frac{1}{2}(\cos(x)(e^{y} - e^{-y}) + i\sin(x)(e^{-y} - e^y)) 21​(cos(x)(ey−e−y)+isin(x)(e−y−ey))

12(cos⁡(x)(ey−e−y)−isin⁡(x)(ey−e−y))\frac{1}{2}(\cos(x)(e^{y} - e^{-y}) - i\sin(x)(e^y - e^{-y})) 21​(cos(x)(ey−e−y)−isin(x)(ey−e−y))

cos⁡(x)sinh⁡(y)−isin⁡(x)sinh⁡(y)\cos(x)\sinh(y) - i\sin(x)\sinh(y) cos(x)sinh(y)−isin(x)sinh(y)

Let's Frame it with a MD quote like this:

cos⁡(z)=cos⁡(x)sinh⁡(y)−isin⁡(x)sinh⁡(y)\cos(z) = \cos(x)\sinh(y) - i\sin(x)\sinh(y) cos(z)=cos(x)sinh(y)−isin(x)sinh(y)

This is saying the following:

  • If I go in the xxx direction, then we are introducing oscillation in the real component and the imaginary component of the number.
  • If I go in the yyy direction, then I am introducing an exponential growth/decay into the real and the complex component.
