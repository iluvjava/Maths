Bootstrap: 1979 by Bradley Elfon

Non-Parametric Bootstrap

Let the samples we have to be the ground truth distribution, this is the assumption we made.

Data set denoted as Dn\mathcal{D}_nDn​.

We assume that the samples we collected are "i.d.d".

Then we sample from the sample Dn\mathcal{D}_nDn​, bbb times, with replacement for each of the sample.

Then we produce multiple estimator from each of the boostrap sample, Dn(b)\mathcal{D}_n^{(b)}Dn(b)​.

Then let that boostrap estimator to be f^(b)(x)\hat{f}^{(b)}(x)f^​(b)(x), and this will be the the set of estimator we get that can be used to produce more predictors of this kind.

Pros and cons

  • Pros

    • Easy to use and applies to everything.

  • Cons

    • Potentially computationally intensive.
    • The reliability is dependent on the convergence rate of the Empirical CDF to the ground truth CDF, and most cases, this is unknown.
