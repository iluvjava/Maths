Intro

In the derivation of a Bayes Classifier, we were able to derive model, but we didn't explain why the Sigmoid function were chosen.

And why a linear model is chosen too.

Here we will start with the basics and derive from the ground up to get to the model we obtained earlier.

Bayes Theorem

Let's start with Bayes and then build up to Sigmoid.

Now, suppose that we want to investigate how the event DDD implies HHH. For example, let's say that DDD is a set of symptoms (Asumme this to be a vector in Rn\mathbb{R}^nRn) for cancer, and HHH is actually having a cancer.

This is a very serious matter, so we are going to use math.

P(H∣D)=P(D∣H)P(H)P(D)\mathbb{P}\left(H|D\right) = \frac{ \mathbb{P}\left(D|H\right)P(H) }{\mathbb{P}\left(D\right)} P(H∣D)=P(D)P(D∣H)P(H)​

This is the portion of people who have cancer and have the specific set of symptoms DDD over all the people who have cancer. This Produces the probability of given the set of symptoms the person ends up having the cancer.

Ok, the problem is: It's very unlikely that this very specific set of symptoms occurred in our data set, because the symptoms covers a wide variety and span a very huge space.

Hence, we can't setimate it using Bayes Theorem alone.

The Oddds

Let's consider the odds of having cancer.

The odds: The ratio between something being true over that thing being false.

Mathematically it's:

P(H∣D)P(¬H∣D)\frac{\mathbb{P}\left(H|D\right)}{ \mathbb{P}\left(\neg H|D\right) } P(¬H∣D)P(H∣D)​

Using Bayes theorem on the numerator and the denominator of this thing, we have:

P(H∣D)P(¬H∣D)=P(D∣H)P(H)1P(D)P(D∣¬H)P(¬H)1P(D)=P(D∣H)P(H)P(D∣¬H)P(¬H)\frac{\mathbb{P}\left(H|D\right)}{ \mathbb{P}\left(\neg H|D\right) } = \frac{ \mathbb{P}(D|H)\mathbb{P}(H) \frac{1}{\mathbb{P}(D)} }{ \mathbb{P}(D|\neg H)\mathbb{P}(\neg H)\frac{1}{\mathbb{P}(D)} } = \frac{ \mathbb{P}(D|H)\mathbb{P}(H) }{ \mathbb{P}(D|\neg H)\mathbb{P}(\neg H) } P(¬H∣D)P(H∣D)​=P(D∣¬H)P(¬H)P(D)1​P(D∣H)P(H)P(D)1​​=P(D∣¬H)P(¬H)P(D∣H)P(H)​

Boom, we had it.

The term P(D)\mathbb{P}(D)P(D) is now out of the picture.

Now, let's denote the odds using the symbol O\mathbb{O}O, well then in that case the above formula is going to be:

O(H∣D)=P(D∣H)P(D∣¬H)O(H)\mathbb{O}(H|D) = \frac{ \mathbb{P}\left(D|H\right) }{ \mathbb{P}\left(D|\neg H\right) } \mathbb{O}(H) O(H∣D)=P(D∣¬H)P(D∣H)​O(H)

Let's introduce the linearity into the picture by taking the log on both side which will give us:

log⁡(O(H∣D))=log⁡(P(D∣H)P(D∣¬H))+log⁡(O(H))=wTx+b(1)\log(\mathbb{O}(H|D)) = \log\left( \frac{ \mathbb{P}\left(D|H\right) }{ \mathbb{P}\left(D|\neg H\right) } \right) + \log\left(\mathbb{O}(H)\right) = w^Tx + b \tag{1} log(O(H∣D))=log(P(D∣¬H)P(D∣H)​)+log(O(H))=wTx+b(1)

And notice that, the variable xxx in the linear model is the odds of observing the list of symptoms for the cancer.

Now, let's see an additional property about the odds:

O(H)=P(H)1−P(H)  ⟺  non trivial mathO(H)1+O(H)=P(H)(2)\mathbb{O}(H) = \frac{\mathbb{P}\left(H\right)}{1 - \mathbb{P}\left(H\right)} \underset{\text{non trivial math}}{\iff} \frac{\mathbb{O}(H)}{1 + \mathbb{O}(H)} = \mathbb{P}\left(H\right) \tag{2} O(H)=1−P(H)P(H)​non trivial math⟺​1+O(H)O(H)​=P(H)(2)

Simply the ratio of the times when something is true over the times when that thing is false.

The Birth of the Sigmoid

The odds with hypothesis:

log⁡(O(H∣D))=wTx+b\log(\mathbb{O}(H|D)) = w^Tx + b log(O(H∣D))=wTx+b

O(H∣D)=wTx+b\mathbb{O}(H|D) = w^Tx + b O(H∣D)=wTx+b

We can convert the conditional odds into probability which will be:

P(H∣D)=O(H∣D)1+O(H∣D)=exp⁡(wTx+b)1+exp⁡(wTx+b)=non-trivial math11+exp⁡(wTx+b)\mathbb{P}\left(H|D\right) = \frac{\mathbb{O}(H|D)}{1 + \mathbb{O}(H|D)} = \frac{\exp(w^Tx + b)}{1 + \exp(w^Tx + b)} \underset{\text{non-trivial math}}{=} \frac{1}{1 + \exp(w^Tx + b)} P(H∣D)=1+O(H∣D)O(H∣D)​=1+exp(wTx+b)exp(wTx+b)​non-trivial math=​1+exp(wTx+b)1​

This is the linear model together with the sigmoid function derived.
