Intro

Kernels provides a fast way of computing high dimensional features.

Kernels provides a way to compute implicit features mapping, like the case with the radial Gaussian Basis for SVMs.

A kernel is a function that can take the inner product given 2 instance of samples from the data:

Let K:Rd×Rd↦RK: \mathbb{R}^{d}\times \mathbb{R}^{d} \mapsto \mathbb{R}K:Rd×Rd↦R, then the function KKK will be a kernel function when:

K(x1,x2)=⟨ϕ(x)ϕ(x2)⟩K(x_1, x_2) = \langle \phi(x) \phi(x_2)\rangle K(x1​,x2​)=⟨ϕ(x)ϕ(x2​)⟩

Where the function ϕ(x)\phi(x)ϕ(x) is a mapping from the lower dimensional feature space to a higher dimensional feature space.

This is the kernel Trick:

And the number of features grow exponentially fast.

K(u,v)=(uTv)dK(u, v) = (u^Tv)^d K(u,v)=(uTv)d
