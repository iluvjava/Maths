Intro

Boosting is a generic algorithm other than a specific model.

Key Idea:

  1. Models are trained sequentially
  2. The next model will have to focus more on the weakness of the previous model.

And this is a measure on the weakness of CCC

Variables:

  • WWW is n×1n\times 1n×1, representing the weights of each sample.
  • EEE is the performance of the model.
  • YYY is n×kn\times kn×k, where nnn is the number of samples and kkk is the number of labels. This is a multiclass classification[1] problem.
  • XXX is the row data matrix, which is n×dn\times dn×d.

Observations:

  • The error rate is close to one if the weight of correctly classified sample are very high.
  • If the heavily weighted samples are not correctly answered, the value of EEE is small.
