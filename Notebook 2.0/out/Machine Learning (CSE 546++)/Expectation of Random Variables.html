Here, we learn how to deal with the expectation of random variables.

Intro

The discrete and the continuous case is analogous to each other, here we will only talk about the discrete case. (Replace Sums with Integrals)

Suppose that random variable XXX has a PMF: fX(x)f_X(x)fX​(x), then the expected value will be defined as:

It's A Linear Operator:

E[aX+b]=aE[X]+b\mathbb{E}\left[aX + b\right] = a \mathbb{E}\left[X\right] + b E[aX+b]=aE[X]+b

It's still Linear wrt multiple Random Variables:

If the random variables X,YX, YX,Y are independent, then:

E[XY]=E[X]E[Y]\mathbb{E}\left[XY\right] = \mathbb{E}\left[X\right]\mathbb{E}\left[Y\right] E[XY]=E[X]E[Y]

Expectation and Variance

The variance of a random variable can be measure by some fancy magics with the Expectation value of the random variable:

E[X2]+E[X]2\mathbb{E}\left[X^2\right] + \mathbb{E}\left[X\right]^2 E[X2]+E[X]2

And hence, it's not hard to see that:

Var[X]=E[(X−E[X])2]=E[X2]+E[X]2\text{Var}\left[X\right] = \mathbb{E}\left[(X - \mathbb{E}\left[X\right])^2\right] = \mathbb{E}\left[X^2\right] + \mathbb{E}\left[X\right]^2 Var[X]=E[(X−E[X])2]=E[X2]+E[X]2

And hence it's not hard to see the property that:

Var[aX+b]=a2Var[X]\text{Var}\left[aX + b\right] = a^2 \text{Var}\left[X\right] Var[aX+b]=a2Var[X]
