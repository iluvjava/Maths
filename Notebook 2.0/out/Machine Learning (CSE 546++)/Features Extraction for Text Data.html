World 2 Vec

I call this a classification approach:

Let's suppose that we have all the English words.

Let's suppose that we were also given English Corpus for training. In the form of sentences of words.

Let the data be {xi,yi}i\{x_i, y_i\}_i{xi​,yi​}i​ denoting the occurrence of words xix_ixi​ and the word next to xix_ixi​, which is yiy_iyi​.

The embedding is obtained by using the latent space of some deep Neural Network.

The input of the Neural Network is a Canonical Basis vector eje_jej​ representing a world in the dictionary. And the output is the next word yiy_iyi​, trained under regression loss would be ok. The embedding is the latent space of the Neural Network, choose whatever you want but make sure it's smaller than the input space of xix_ixi​.

Train Deep Neural Net f,gf, gf,g such that:

min⁡f,g∑i=1N∥yi−f(g(xi))∥22\min_{f, g} \sum_{i = 1}^{N}\Vert y_i - f(g(x_i))\Vert_2^2 f,gmin​i=1∑N​∥yi​−f(g(xi​))∥22​

Then the embedding are y=g(xi)y = g(x_i)y=g(xi​).

Using this embedding, one can predict the probability of a word xix_ixi​ being close to xjx_jxj​ using:

softmax(g(xi)Tg(xj))\text{softmax}\left( g(x_i)^Tg(x_j) \right) softmax(g(xi​)Tg(xj​))

This is why I call this a classification approach.

But the embedding is useful by itself.

LDA and NMF

NMF: Non-Negative Matrix Factorization

These methods are used for topics identification over topic labels for each document.

Here we assume that a distribution of certain words in the dictionary represents a topic.

Let the matrix XXX be the data matrix where each row represent each document and the content of each row is the distribution of dictionary words.

X[i,j]=#time word xj appeared in doc diTotal words in diX[i, j] = \frac{\#\text{time word } x_j \text{ appeared in doc } d_i}{\text{Total words in }d_i} X[i,j]=Total words in di​#time word xj​ appeared in doc di​​

Consider the following matrix decomposition that minimizes the squared errors on the given data matrix.

arg⁡min⁡W≥0,H≥0∥X−WHT∥22\arg\min_{W\ge\mathbf{0},H\ge\mathbf{0}} \Vert X - WH^T\Vert_2^2 argW≥0,H≥0min​∥X−WHT∥22​

The problem is to look for a factorization of matrix XXX by 2 positive matrices, WWW, representing the weight of each topics, and columns in HHH representing a topic for all the documents.

Using the otter product interpretation, the columns of WWW is used to span the columns of matrix XXX, and the column of Matrix X represents the distribution of a particular word or different documents.

My Best Interpretation:

X[i,:]=W[i,:]HTX[i,:] = W[i,:]H^TX[i,:]=W[i,:]HT or X[i,:]T=HW[:,i]X[i, :]^T = HW[:,i]X[i,:]T=HW[:,i] reconstructs words distribution of a particular document in the sample. The iii th row of the WWW matrix represent a linear combinations of the rows of HHH, which is a typical word distribution of a given topic.

Note, The problem becomes equivalent to k-mean cluster if HHT=IHH^T = IHHT=I constraints are added. [1]
