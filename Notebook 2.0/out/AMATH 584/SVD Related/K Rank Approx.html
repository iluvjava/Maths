The Rank Approximation theorem is also stated as the Eckart Young theorem. prepreq: [[SVD]]

The cool thing here is, cropping off the some of the singular values from an SVD decomposition of a matrix is not going to ruin the matrix by a lot, and most of the information is going to be preserved.

Wiki: Low-Rank Approximation

So What is going on here?

Given any matrix, we figure out the singular value decomposition of such a matrix. Which is getting that A=UΣVTA = U\Sigma V^TA=UΣVT for any matrix.

Given that all the singular values in Σ\SigmaΣ are sorted in descending order, we have the choice reduce the singular values and crop of the corresponding columns in UUU and rows inVTV^TVT.

This is reducing the rank of the matrix ATAA^TAATA and AATAA^TAAT. By the way, the rank of the matrix AAA has the following:

rank(A)≤min⁡(rank(ATA),rankAAT)\text{rank}(A) \leq \min(\text{rank}(A^TA), \text{rank}AA^T) rank(A)≤min(rank(ATA),rankAAT)

And this, is also the solution to the following problem:

Given a matrix AAA, we want to find another matrix A′A'A′ with rank less than the matrix AAA, so that, it's the closest approximation to the matrix AAA.

Let's denote the matrix A∈Rm×nA\in \mathbb{R}^{m\times n}A∈Rm×n to be the matrix which we are trying to approximate using the SVD: A=UΣVTA = U\Sigma V^TA=UΣVT

Then the number of non-zero singular value will determine the size of the matrix Σ\SigmaΣ, in this case the size will be k=min⁡(m,n)k = \min(m, n)k=min(m,n).

The approximated expression can be expressed as $$\hat{A} = \hat{U}\hat{\Sigma}\hat{V}^H$$

After the rank approximation with the first ppp principal values, we have to make sure that the size of A^\hat{A}A^ and AAA stays unchanged. Hence we are doing the following things:

  • Keep the first ppp columns of matrix UUU
  • Keep the first ppp rows and column of matrix Σ\SigmaΣ
  • Keep the first ppp rows of matrix VTV^TVT, or equivalently the first ppp columns of VVV

So the mapping before the approximation is going to be:

Rn×k↦Rk×k↦Rk×m\mathbb{R}^{n\times k} \mapsto \mathbb{R}^{k\times k} \mapsto\mathbb{R}^{k\times m} Rn×k↦Rk×k↦Rk×m

And then the reduced mapping is going to be:

Rn×p↦Rp×p↦Rp×m\mathbb{R}^{n\times p} \mapsto \mathbb{R}^{p\times p} \mapsto\mathbb{R}^{p\times m} Rn×p↦Rp×p↦Rp×m

Then we can confidently say that:

A^=argminrank(M)≤p(∥A−M∥F)\hat{A} = \text{argmin}_{rank(M) \leq p}\left( \Vert A - M \Vert_F\right) A^=argminrank(M)≤p​(∥A−M∥F​)
