Intro

The Krylov subspace is spanning the same space as the QQQ matrix of the Hessenberg Similarity Transform.

The Arnoldi Iterations assert this fact, and it can be consider the that Krylov Matrix written as the following:

Kn=[bAbAb2⋯An−1b]K_n = \begin{bmatrix} \\[1em] b & Ab & Ab^2 & \cdots A^{n - 1}b \\[1em] \\ \end{bmatrix} Kn​=⎣⎡​b​Ab​Ab2​⋯An−1b​⎦⎤​

Is spanned by the orthogonal subspace:

Kn=QnRnK_n = Q_nR_n Kn​=Qn​Rn​

The Krylov matrix is given by the product of the QQQ matrix from the Arnoldi Iterations and the RRR Matrix can be computed after we produced the QQQ matrix from the Arnoldi Iterations.

Warning:

Usually, KnK_nKn​ is very numerically unstable, and hence we only use the matrix in theoretical analysis, not actual implementation of numerical algorithm.

Observation:

Representing bbb using eigenvectors of AAA, and this KnK_nKn​ will tell use a lot about the Eigenvectors of the matrix AAA.

Claim 1

The matrix H~n\tilde{H}_{n}H~n​ from the Arnoldi iterations either by Modified GS, or the Householder Triangularization, will have the same eigenvector as the matrix AAA.
