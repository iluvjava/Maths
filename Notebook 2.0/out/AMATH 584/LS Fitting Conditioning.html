Theorem 18.1 (Trefethen)

This theorem is going to be cited directly from Trefethen's book on Numerical LInear Algebra:

κb↦y=1cos⁡θ\underset{b\mapsto y}{\kappa} = \frac{1}{\cos{\theta}} b↦yκ​=cosθ1​

This describes the conditioning when be pertubated the value bbb and how it affects the yyy on the output space of the problem.

This is saying that, if the vector bbb is perpendicular to the range of the matrix, then, a tiny pertubations on the vector b is going to cause big changes on the projection of the vector. (namely from zero to something that is nonzero). And this can happens in just right way that, catastrophic cacellation errors creeps into the system.

Note: it's linear, and therefore, combining the condition number is able to described composite pertubations on the inputs space of the function affecting the outputs of the function.
