Looking for sparse representation, and it's application in signal processing.

[[Regularization]]

Intro

We are interested in reconstructing the signal with some sub-sampling, or deblur, or, denoise the signal.

The key is to look for a sparse representation of the image/signals under some basis.

Suppose we sub-sample a signal, and then we are trying to reconstruct the signal using as few frequencies as possible.

Step 1:

The function is a representation of some known basis, and it's compressible under that basis.

f=Ψcf = \Psi c f=Ψc

Where the basis matrix is Ψ\PsiΨ and the ccc vector are the coefficients, they are going to represent the loading of the basis vector.

The ccc vector is sparse, mostly zero, representing the fact that our function is compressible under the basis Ψ\PsiΨ.

Step 2:

b=Φfb = \Phi f b=Φf

Let bbb be the sub-sampling vector, by which we mean that, it's a down-sample of the given signal fff, and the matrix Φ\PhiΦ, is a matrix that is mostly zeros, with few ones as its entry. This matrix down-sample the signal f.

Step 3

The down sample vector can be represented as:

b=Φf=ΦΨc=Acwhere: A=ΦΨb = \Phi f = \Phi \Psi c = Ac \quad \text{where: } A = \Phi\Psi b=Φf=ΦΨc=Acwhere: A=ΦΨ

This is an basic Linear algebra problem.

So the question we are answering is:

  • How do I represent the signal using as little basis vector as possible such that if I down sample the signal, I will get what I observed to be (The bbb vector)?
