Intro

Previous we were exposed to the idea of conjugate functions. And we learn the important facts that:

  1. The dual variable is the variable of the conjugate function.

  2. The double conjugate function of fff is itself when the function is closed and convex.

  3. We know how to figure out the conjugate of constraint functions, of the form: Cx=bCx = bCx=b, Cx≤bCx \le bCx≤b, where CCC is a matrix, and this will represent the feasible domain for the optimization problem.

  4. One of the important lemma that can be concisely stated as:

min⁡xmax⁡yF(x,y)≥max⁡ymin⁡xF(x,y)\min_x{\max_y F(x, y)} \ge \max_y\min_xF(x,y) xmin​ymax​F(x,y)≥ymax​xmin​F(x,y)

Which is pretty useful for the analysis of Duality.

For Duality, we will be interested about the following perspectives (Views, things, interpretations)

  1. Computational benefits of looking for the dual problem.
  2. Perturbation and sensitivity analysis.
  3. Weak and strong duality.
  4. Characterizing the optimality of the solutions.

Equality Constraints, Any Objective

Consider objective function g(x)g(x)g(x) and equality constraints: Cx=dCx = dCx=d, where CCC is a matrix. We are going to look for the dual of the problem.

Then the objective function can be mixed with the constraint function, giving us:

min⁡x{g(x)+δ0(Cx−d)}(1)\min_x \left\lbrace g(x) + \delta_0(Cx - d) \right\rbrace \tag{1} xmin​{g(x)+δ0​(Cx−d)}(1)

Take notice that, the function δ0\delta_0δ0​ can be written as the conjugate conjugate (Because the feasible region defined is a linear subspace, and that is convex):

δ0(Cx−d)=sup⁡z{zT(Cx−d)−δ0∗(z)}\delta_0 (Cx - d) = \sup_z \left\lbrace z^T(Cx - d) - \delta_0^*(z) \right\rbrace δ0​(Cx−d)=zsup​{zT(Cx−d)−δ0∗​(z)}

And remember from the Second part of the duality preparation, the conjugate of the zero indicator function is actually zero, and therefore, we have:

δ0(Cx−d)=sup⁡z{zT(Cx−d)}\delta_0 (Cx - d) = \sup_z \left\lbrace z^T(Cx - d) \right\rbrace δ0​(Cx−d)=zsup​{zT(Cx−d)}

And then we will have another expression for (1), which is:

min⁡x{sup⁡z{g(x)+zT(Cx−d)}}(2)\min_x \left\lbrace \sup_z \left\lbrace g(x) + z^T(Cx - d) \right\rbrace \right\rbrace \tag{2} xmin​{zsup​{g(x)+zT(Cx−d)}}(2)

Amazingly, the thing that is inside the min sup operator has the name: Lagrangian. Which has information for both the primal and dual problem.

From (2), we can swap the sup and min operator, which will give us the inequality that:

(2)≥sup⁡z{−zd+min⁡x{g(x)+zTCx}}(2) \ge \sup_z \left\lbrace -zd + \min_x \left\lbrace g(x) + z^TCx \right\rbrace \right\rbrace (2)≥zsup​{−zd+xmin​{g(x)+zTCx}}

Notice that I also take out the −zd-zd−zd term because it doesn't depend on the variable x. And then I can also take the negative inside and chang the min to a supsupsup, giving us:

sup⁡z{−zd−sup⁡x{−g(x)−zTCx}}\sup_z \left\lbrace -zd - \sup_x \left\lbrace -g(x) - z^TCx \right\rbrace \right\rbrace zsup​{−zd−xsup​{−g(x)−zTCx}}

Immediately observe the fact that we are having:

sup⁡x{−g(x)−zTCx}=g∗(−CTz)\sup_x \left\lbrace -g(x) - z^TCx \right\rbrace = g^*(-C^Tz) xsup​{−g(x)−zTCx}=g∗(−CTz)

This will in fact gives the fact that:

sup⁡z{−zd−g∗(−CTz)}(3)\sup_z \left\lbrace -zd - g^*(-C^Tz) \right\rbrace \tag{3} zsup​{−zd−g∗(−CTz)}(3)

(3) is the dual problem of (1), in this particular case.

Inequality Constraints

Consider the primal problem phrased by:

min⁡x(g(x))s.tCx≤d\min_x (g(x)) \quad s.t \quad Cx \le d xmin​(g(x))s.tCx≤d

And that is just:

min⁡x{g(x)+sup⁡z{zT(d−Cx)−δ+∗(z)}}\min_x \left\lbrace g(x) + \sup_z \left\lbrace z^T(d- Cx) - \delta_+^*(z) \right\rbrace \right\rbrace xmin​{g(x)+zsup​{zT(d−Cx)−δ+∗​(z)}}

And this is the way to figure out the duality theory for linear programming. Isn't that cool?

A Summarization

What we did here all the time, it's just using the property of the double conjugacy:

h(x)=sup⁡z{xTz−h∗(z)}h(x) = \sup_z \left\lbrace x^Tz - h^*(z) \right\rbrace h(x)=zsup​{xTz−h∗(z)}

Where the function is closed and convex. This is a very general way to compute the dual, and it's for the general case too.

**Linear Programming Duality **

At this stage, is not hard to see that (4) is applicable for duality in linear programming.

The conjugate of a linear objective function is the indicator function. If f(x)=CTxf(x) = C^Txf(x)=CTx, then the conjugate is δ0(z−c)\delta_0(z - c)δ0​(z−c). When z=cz = cz=c, this is good, when it's not, this is not good.

Complex Example

A general example will be:

min⁡x{sup⁡z{g(x)+cTx+zT(Ax−b)}}\min_x \left\lbrace \sup_z \left\lbrace g(x) + c^Tx + z^T(Ax - b) \right\rbrace \right\rbrace xmin​{zsup​{g(x)+cTx+zT(Ax−b)}}

≥sup⁡z{−zTb−h∗(z)+min⁡x{g(x)+xT(ATz+c)}}\ge \sup_z\left\lbrace -z^Tb - h^*(z) + \min_x \left\lbrace g(x) + x^T(A^Tz + c) \right\rbrace \right\rbrace ≥zsup​{−zTb−h∗(z)+xmin​{g(x)+xT(ATz+c)}}

And that is the dual, maximal system.
