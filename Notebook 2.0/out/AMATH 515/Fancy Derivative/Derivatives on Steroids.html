Ax Composition

let AAA be a m×nm\times nm×n matrix, let fff be a Rm↦R\mathbb{R}^m \mapsto \mathbb{R}Rm↦R function, then taking the gradient on f(Ax)f(Ax)f(Ax) will be like:

∇[f(Ax)]=AT∇f(Ax)(1)\nabla [f(Ax)] = A^T \nabla f(Ax) \tag{1} ∇[f(Ax)]=AT∇f(Ax)(1)

And the Hessian of the function will be like:

∇2[f(Ax)]=AT∇f(Ax)A(2)\nabla^2 [f(Ax)] = A^T \nabla f(Ax) \tag{2}A ∇2[f(Ax)]=AT∇f(Ax)A(2)

Proof is investigated in HW1 for the class.

Example: Poisson Regression with Regularization

Consider the scalar function:

f(x):=∑i=1m(exp⁡(aiTx)−bi(aiTx))+λ∥x∥22f(x):= \sum_{i = 1}^{m} \left( \exp(a_i^Tx) - b_i (a_i^Tx) \right) + \frac{\lambda\Vert x\Vert^2}{2} f(x):=i=1∑m​(exp(aiT​x)−bi​(aiT​x))+2λ∥x∥2​

Which is a common type of regularized regression problem, in this case, we are looking for the gradient and the Hessian of the objective function.

Firstly, notice that the function consists of 2 parts, the first part is a composite function involving a matrix vector product and the second part consists of the regularization term. Therefore we could have:

g(Ax):=∑i=0m(exp⁡(aiTx)−bi(aiTx))g(Ax) := \sum_{i=0}^{m} \left( \exp(a_i^Tx) - b_i (a_i^Tx) \right) g(Ax):=i=0∑m​(exp(aiT​x)−bi​(aiT​x))

g(x)=∑i=0m(exp⁡(xi)−bixi)g(x) = \sum_{i = 0}^{m} \left( \exp(x_i) - b_ix_i \right) g(x)=i=0∑m​(exp(xi​)−bi​xi​)

∇g(x)=exp⁡E(x)−b\nabla g(x) = \exp^E(x) - b ∇g(x)=expE(x)−b

Where exp⁡E(x)\exp^E(x)expE(x) is element wise operations on an array, which is like:

exp⁡E(x)=[exp⁡(x1)exp⁡(x2)⋮exp⁡(xm)]\exp^E(x) = \begin{bmatrix} \exp(x_1) \\ \exp(x_2) \\ \vdots \\\exp(x_m) \end{bmatrix} expE(x)=⎣⎢⎢⎢⎢⎡​exp(x1​)exp(x2​)⋮exp(xm​)​⎦⎥⎥⎥⎥⎤​

And therefore, the gradient of fff is gonna be like:

∇f(x)=∇[g(Ax)]+∇[λ∥x∥22]\nabla f(x) = \nabla [g(Ax)] + \nabla[\lambda\frac{\Vert x\Vert^2}{2}] ∇f(x)=∇[g(Ax)]+∇[λ2∥x∥2​]

∇f(x)=AT(exp⁡E(Ax)−b)+λx\nabla f(x) = A^T(\exp^E(Ax) - b) + \lambda x ∇f(x)=AT(expE(Ax)−b)+λx

And it's not hard to see that, taking the Jacobian on the function ∇g(x)\nabla g(x)∇g(x) will yields a diagonal matrix because each of the element of the vector only relates to one of the variable, and therefore, we have:

∇2g(x)=diag(exp⁡E(x))\nabla^2 g(x) = \text{diag}(\exp^E(x)) ∇2g(x)=diag(expE(x))

And applying the rules from (1), we have:

∇2[f(x)]=ATdiag(exp⁡E(Ax))A+λI\nabla^2[f(x)] = A^T\text{diag}(\exp^E(Ax))A + \lambda I ∇2[f(x)]=ATdiag(expE(Ax))A+λI

Example: Logistic Regression with Regularization

f(x):=∑i=1m(ln⁡(1+exp⁡(aiTx))−bi(aiTx))+λ∥x∥22f(x):= \sum_{i = 1}^{m} \left( \ln(1 + \exp(a_i^T x)) - b_i(a_i^Tx) \right) + \frac{\lambda\Vert x\Vert^2}{2} f(x):=i=1∑m​(ln(1+exp(aiT​x))−bi​(aiT​x))+2λ∥x∥2​

Notice that the summation part can be an implicit composition function:

g(Ax)=∑i=1m(ln⁡(1+exp⁡(aiTx)))+bTAxg(Ax) = \sum_{i = 1}^{m} \left( \ln(1 + \exp(a_i^T x)) \right) + b^TAx g(Ax)=i=1∑m​(ln(1+exp(aiT​x)))+bTAx

So then in this case, we can use the Ax composition rule and get the expression as:

∇[g(Ax)]=AT∇g(Ax)=ATexp⁡(Ax)1+exp⁡(Ax)+ATb\nabla [g(Ax)] = A^T \nabla g(Ax) = A^T \frac{\exp(Ax)}{1 + \exp(Ax)}+ A^Tb ∇[g(Ax)]=AT∇g(Ax)=AT1+exp(Ax)exp(Ax)​+ATb

And hence with regularization, the gradient of the function is just:

∇f(x)=AT∇g(Ax)=ATexp⁡(Ax)1+exp⁡(Ax)+ATb+λx\nabla f(x) = A^T \nabla g(Ax) = A^T \frac{\exp(Ax)}{1 + \exp(Ax)}+ A^Tb + \lambda x ∇f(x)=AT∇g(Ax)=AT1+exp(Ax)exp(Ax)​+ATb+λx

And the Hessian of the function will be:

∇2f(x)=ATdiag(exp⁡E(x)(1+exp⁡E(x))2)A+λI\nabla^2 f(x) = A^T \text{diag}\left( \frac{\exp^E(x)}{(1 + \exp^E(x))^2} \right)A + \lambda I ∇2f(x)=ATdiag((1+expE(x))2expE(x)​)A+λI

High Dimensional Composition

Composition type:

(f:Rm↦R)↦(G:Rn↦Rm)(f:\mathbb{R}^m \mapsto \mathbb{R})\mapsto(G:\mathbb{R}^n \mapsto \mathbb{R}^m) (f:Rm↦R)↦(G:Rn↦Rm)

Then the composition of f∘Gf\circ Gf∘G is gonna be like:

∇G(x)T∇f(G(x))\nabla G(x)^T\nabla f(G(x)) ∇G(x)T∇f(G(x))

The Jacobian transposed multiplied by Gradient of F with G. The easiest case for G(x)G(x)G(x) is AAA, and the Jacobian of AxAxAx is just AAA, and hence, the Formula we already gotten.

The Hessian will be a different story because we need product rule here.
