The steepest descend method for solving the matrix vector equation. Basic understanding about [[Hermitian Adjoint]] is needed for exploiting the structure of symmetric matrix using steepest Descend.

Intro

Our objective is to solve:

Ax=bAx = b Ax=b

Where we assume that the matrix AAA is Symmetric Positive Definite[1] and sparse, and we treat it as a black box transformation applied to vectors.

Consider the function:

f(x)=12xTAx−bTx+c∇x[f(x)]=12(AT+A)x−b  ⟹  (1)∇x[f(x)]=Ax−bLet: ∇x[f(x)]=0  ⟹  Ax=b(1)\begin{aligned} f(x) & =\frac{1}{2}x^TAx - b^T x + c \\ \nabla_x[f(x)] &= \frac{1}{2}(A^T + A)x - b \\ \underset{(1)}{\implies} \nabla_x[f(x)] &= Ax - b \\ \text{Let: }\quad \nabla_x[f(x)] &= 0 \\ \implies Ax & = b \end{aligned} \tag{1} f(x)∇x​[f(x)](1)⟹​∇x​[f(x)]Let: ∇x​[f(x)]⟹Ax​=21​xTAx−bTx+c=21​(AT+A)x−b=Ax−b=0=b​(1)

(1): Using that assumption that matrix AAA is orthogonal.

Residual is also the gradient at that particular step.

Observe

The residual is the gradient of the function.

If the matrix is not symmetric, then the gradient is not the residual at each teration.

The problem we are solving has nothing to do with minimizing the 2 norm of Ax−bAx - bAx−b. It's only for the symmetric matrix AAA. It's minimzing the energy norm, if AAA is symmetric positive definite.

arg⁡min⁡x∥x∥A+xTb\arg \min_x \Vert x\Vert_A + x^Tb argxmin​∥x∥A​+xTb

If the matrix is not positive definite, then it has null space, potentially ill-conditioned or not invertible.

Moral of the Story
