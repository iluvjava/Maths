[[Multi-Dimensional PDEs, Schordinger Eqn in Box]] Here, we are sticking with the same thing, but we are going to look into some concrete examples for it.

Intro

We are interested in a sound waves in a rectangular cavity.

The system of equations is given like:

∂t2[ψ]=α2∇2ψ\partial_t^2[\psi] = \alpha^2\nabla^2\psi ∂t2​[ψ]=α2∇2ψ

And ψ\psiψ is the displacement.

And the box is rectangular.

ψ=0when: x=0,L1y=0,L2z=0,L3\psi = 0 \quad \text{when: } x = 0, L_1\quad y = 0, L_2\quad z = 0, L_3 ψ=0when: x=0,L1​y=0,L2​z=0,L3​

Separation of Variables

The assumption for the solution format is:

ψ(x,y,z,t)=X(x)Y(y)Z(z)T(t)(1)\psi(x, y, z, t) = X(x)Y(y)Z(z)T(t) \tag{1} ψ(x,y,z,t)=X(x)Y(y)Z(z)T(t)(1)

Substituting it back to the original equation yields

T′′XYZ=α2(X′′YZT+XY′′XT+XYZ′′T)T''XYZ = \alpha^2(X''YZT + XY''XT + XYZ''T) T′′XYZ=α2(X′′YZT+XY′′XT+XYZ′′T)

Dividing both size by XYXTXYXTXYXT, we have:

T′′T=α2(X′′X+Y′′Y+Z′′Z)\frac{T''}{T} = \alpha^2 \left( \frac{X''}{X} + \frac{Y''}{Y} + \frac{Z''}{Z} \right) TT′′​=α2(XX′′​+YY′′​+ZZ′′​)

and, spoiler alert, because this is true for all value of x,y,z,tx, y, z, tx,y,z,t, the left hand side and the RHS will have to be a constant. But there will be many constants for us to locate the basis function in the spatial domain.

Note: The α\alphaα is going to be with TTT, on the left hand side of the equation.

And we are going to make the assumption that the constant on the RHS is −λ2-\lambda^2−λ2, this is true because we are looking at dirichlet boundary conditions for the system.

Applying the SOV, then we will have:

X′′X+Y′′Y+X′′Z=−λ2\frac{X''}{X} + \frac{Y''}{Y} + \frac{X''}{Z} = -\lambda^2 XX′′​+YY′′​+ZX′′​=−λ2

Then, we will know that:

X′′X=−a2Y′′Y=−b2X′′Z=−c2(2)\frac{X''}{X} = -a^2 \quad \frac{Y''}{Y} = -b^2 \quad \frac{X''}{Z} = -c^2 \tag{2} XX′′​=−a2YY′′​=−b2ZX′′​=−c2(2)

Well in this case, we will be getting basis functions for X,Y,ZX, Y, ZX,Y,Z, and each of then will be identified with a natural numbers. Which determines the frequencies of the Basis function.

Notice that it's hinted that:

−λm,n,l2=−(an2+bm2+cl2)-\lambda^2_{m, n, l} = -(a^2_n + b^2_m + c^2_l) −λm,n,l2​=−(an2​+bm2​+cl2​)

Combining what I just said about a lot of solutions each of them being identified with a natural number, we will have:

an=nπL1bm=mπL2cl=lπL3a_n = \frac{n\pi}{L_1} \quad b_m = \frac{m\pi }{L_2}\quad c_l = \frac{l\pi}{L_3} an​=L1​nπ​bm​=L2​mπ​cl​=L3​lπ​

And then those are the frequencies for basis function X,Y,ZX, Y, ZX,Y,Z, and the basis function is sin⁡\sinsin hence the solution will be in the form of:

un,m,l(x,y,z,t)=sin⁡(nπxL1)sin⁡(mπyL2)sin⁡(lπzL3)Tm,n,l(t)u_{n, m, l}(x, y, z, t) = \sin \left( \frac{n\pi x}{L_1} \right)\sin\left( \frac{m\pi y}{L_2} \right)\sin\left( \frac{l\pi z}{L_3} \right)T_{m, n, l}(t) un,m,l​(x,y,z,t)=sin(L1​nπx​)sin(L2​mπy​)sin(L3​lπz​)Tm,n,l​(t)

Amazing.

λm,n,l2=π2((nL1)2+(nL2)2+(nL3)2)w\lambda_{m, n, l}^2 = \pi^2\left( \left( \frac{n}{L_1} \right)^2 + \left( \frac{n}{L_2} \right)^2 + \left( \frac{n}{L_3} \right)^2 \right)w λm,n,l2​=π2((L1​n​)2+(L2​n​)2+(L3​n​)2)w

The Time domain

Tm,n,l′′+α2λm,n,l2Tm,n,l=0T''_{m, n, l} + \alpha^2\lambda_{m, n, l}^2T_{m, n, l} = 0 Tm,n,l′′​+α2λm,n,l2​Tm,n,l​=0

Then it can be said that:

Tn,m,l=Asin⁡(wn,m,lt)+Bcos⁡(wn,m,lt)T_{n, m, l} = A\sin(w_{n, m, l}t) + B\cos(w_{n, m, l}t) Tn,m,l​=Asin(wn,m,l​t)+Bcos(wn,m,l​t)

Where

wn,m,l=απ(nL1)2+(mL2)2+(lL3)2w_{n, m, l} = \alpha\pi \sqrt{ \left( \frac{n}{L_1} \right)^2 + \left( \frac{m}{L_2} \right)^2 + \left( \frac{l}{L_3} \right)^2 } wn,m,l​=απ(L1​n​)2+(L2​m​)2+(L3​l​)2​

Notice that the frequencies of the sound wave is quantized.

The lowest frequency will be given by w1,1,1w_{1, 1, 1}w1,1,1​

Some Important Facts

This means that, for higher requencies, it cannot be written as an integer multiple of w1,1,1w_{1, 1, 1}w1,1,1​.

Notice that, in the first dimension, the harmonics of the basis function is given as:

wn=nαπLw_n = \frac{n\alpha\pi}{L} wn​=Lnαπ​

Which means that w2w_2w2​ is a multiple of w4w_4w4​, it's like a combination of ordered harmonics.

In the case of 3D partial standing waves, the harmonics is different, and they are not 2x multiple of each other.

That is also the reason why some of the instruments that make standing waves on a string has more of a soothing affect, it's because of the properties of the harmonics on the 1D vibrating string.
