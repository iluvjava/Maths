Intro

We are going to solve the PDEs in the cylindrical coordinates.

The Helmholtz equation is the SPATIAL part of the wave equation.

And, the given PDE is in this form:

∇2⋅u=−λ2∂t2[u]\nabla^2\cdot u = -\lambda^2 \partial_t^2[u] ∇2⋅u=−λ2∂t2​[u]

Here is the parameters for the cylinder:

  1. Cylinder radius: aaa
  2. Radial direction of: rrr
  3. Angular dependence: θ\thetaθ
  4. Height: LLL
  5. Height Dependence: zzz

This is like a flute. The solution is going to satisfy the wave equation, and it's going to represent the waves inside of the cylinder.

This is the boundary conditions for the equation:

u=0z=0,Lu = 0 \quad z = 0, L u=0z=0,L

There is no vibration on the cap of the cylinder.

u=0r=au = 0 \quad r = a u=0r=a

There is no vibration on the wall of the cylinder.

Here is a summary on the procedures to take when solving this equation:

  1. Separate it into spatial domain and time domain, let the function takes the form: ϕ(x,y,z)T(t)\phi(x, y, z)T(t)ϕ(x,y,z)T(t)
  2. Then for the spatial domain, using the separation of variables to get the solution for each of the function that governs each dimension in the Cylindrical coordinate, by assuming that ϕ(x,y,z)=Z(z)Θ(θ)R(r)\phi(x, y, z) = Z(z)\Theta(\theta)R(r)ϕ(x,y,z)=Z(z)Θ(θ)R(r).
  3. Solve the ODE, starting with the part involving only Z(z)Z(z)Z(z) using the boundary conditions that there is no vibrations on the cap of the cylinder, then Θ(θ)\Theta(\theta)Θ(θ). And the function Θ(θ)\Theta(\theta)Θ(θ) has a 2π2\pi2π period to it. Then finally, solve the R(r)R(r)R(r) which is the bessel's equation, extract the bessel's of the first kind as the solution for functions R(r)R(r)R(r) because of the boundary conditions that R(0)≠inf⁡R(0) \ne \infR(0)​=inf. When solving the bessel's equation, see: [[Bessel's Equation]] for a more formal treatment of the matter.
  4. Then, we get the solutions for all the basis functions, in which wee will also have accumulated all the constants we need, wrt to 2 indices: m,n,lm, n, lm,n,l.
  5. Then, we add the time domain into the picture and solve the ODE involving the time and get the full solution of the system.
