Intro

Markov Inequality

For any t≥0t\ge 0t≥0 and non-negative random variable XXX, we have:

P(X≥t)≤E[X]t\mathbb{P}\left(X \ge t\right) \le \frac{\mathbb{E}\left[X\right]}{t} P(X≥t)≤tE[X]​

Which is equivalent to:

1−P(X≤t)≤E[X]t1 - \mathbb{P}(X\le t) \le \frac{\mathbb{E}\left[X\right]}{t} 1−P(X≤t)≤tE[X]​

And this inequality can be used to bound the MLE.

This is saying that, using the expected value of a random variable, the probability of observing random variable above some threshold is bounded by the Expected value of the random variable.
