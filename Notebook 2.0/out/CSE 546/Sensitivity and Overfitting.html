Shrinkage Properties

As increases the value of the λ\lambdaλ, estimator for the function polynomials gets smaller and smaller.

As we increase the value of λ\lambdaλ, we will goes from over-fitting to underfitting for the given model.

Note: The last estimator that goes to zero is the most important features of the data set.

Here, we will assume that the feature of the data consist of nnn idd variables draw from a unit guassian distribution, then the expected value of the data matrix XXX has the property that:

E[XTX]=nI\mathbb{E}\left[X^TX\right] = n\mathbb{I} E[XTX]=nI

In this case, this is true, and using this property, can figure out the shrinkage property of the ridge regression by saying that:

(XTX+λI)TXTy=w^  ⟹  E[w^]=(n+λ)−1XTy(X^TX + \lambda\mathbb{I})^TX^Ty = \hat{w} \implies \mathbb{E}\left[\hat{w}\right] = (n + \lambda)^{-1}X^Ty (XTX+λI)TXTy=w^⟹E[w^]=(n+λ)−1XTy

Take note that the regress estimator without the regularization is given by:

(XTX)−1XTy  ⟹  w^ls=XTyn(X^TX)^{-1}X^Ty \implies \hat{w}_{\text{ls}} = \frac{X^Ty}{n} (XTX)−1XTy⟹w^ls​=nXTy​

So then, we can relate the ridge regression to the least square estimator and get:

E[w^]=nn+λw^ls\mathbb{E}\left[\hat{w}\right] = \frac{n}{n + \lambda}\hat{w}_{\text{ls}} E[w^]=n+λn​w^ls​

And this is the shrinkage properties of ridge regression demonstrated. In this sense, we will have the value of the esimator appraoches zero as the regularization parameters increases.
