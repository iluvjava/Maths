[[Cone]]


---
### **Intro**

A polar cone, denoted as $K^\circ$ where $K$ is a cone is defined to be the set of all vectors $v$ that makes an obstuse angle with vectors from the set $K$: 

$$
K^\circ := 
\left\lbrace
    v\in \mathbb{R}^n: 
    \langle v, x\rangle \le 0 \; \forall x \in K
\right\rbrace
$$
