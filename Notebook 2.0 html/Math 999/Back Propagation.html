The objective is to find the derivative of the weight matrix given a set of deep Neuro-Network.

1. Intro

Let's be familiar with the set up of a Neuro Network with one hidden layer.

E(x,θ)=12∥f3(W2f2(W1x+b1)+b2)−y∥2E(x, \theta) = \frac{1}{2}\Vert f_3(W_2f_2(W_1x + b_1) + b_2) - y\Vert^2 E(x,θ)=21​∥f3​(W2​f2​(W1​x+b1​)+b2​)−y∥2

And this is the error function of the neuro net. It's taking the error by looking at a single sample.

3 layers, 2 fully connected layers exist between the input and the output, supported by the W1W_1W1​ and the W2W_2W2​ weight matrices. α1,α2\alpha_1, \alpha_2α1​,α2​ are the biases.

f3,f2f_3, f_2f3​,f2​ are th activation function on the second and the third layer of the Neuron.

θ\thetaθ is just denoting all the parameters one can find in the Neuro Net.

Training:

We feed the Neuro Net a small example from the populations, and then we minimize the errors on that smaller patch of the samples. Notice, that convergence is not promised, but we can assume that there exists a function that can approximate the whole populations.

4. Taking Derivative on Entries of Weight the Weight Matrix

The loss function is a scalar function, wrt all the weights and biases in the network.

We are taking the derivative wrt to a particular weight, say wi,j(k)w_{i, j}^{(k)}wi,j(k)​, this is just an ordinary differential because the loss function is just a scalar output as well.

In addition, I just want to know the derivative for one sample, say yyy. The loss function will be like:

∂wi,j(k)L(x(N)∣y)\partial_{w_{i, j}^{(k)}}L\left(x^{(N)}|y\right) ∂wi,j(k)​​L(x(N)∣y)

=∇[L(x(N)∣y)]T∂wi,j(k)fout(p(N))=\nabla[L(x^{(N)}|y)]^T\partial_{w_{i, j}^{(k)}} f_{out}\left(p^{(N)}\right) =∇[L(x(N)∣y)]T∂wi,j(k)​​fout​(p(N))

Here, we will need the gradient of the loss function, using chain rule, and only focus on the second expression we have:

∂wi,j(k)fout(p(N))=fout′(p(N))∘∂wi,j(k)[p(N)]\partial_{w_{i, j}^{(k)}}f_{out}\left(p^{(N)}\right) = f'_{out}\left(p^{(N)}\right)\circ \partial_{w_{i, j}^{(k)}}\left[p^{(N)}\right] ∂wi,j(k)​​fout​(p(N))=fout′​(p(N))∘∂wi,j(k)​​[p(N)]

We are not using the Jacobian because foutf_{out}fout​ is a univariate function that got broadcast to the vector p(N)p^{(N)}p(N). Hence we simply use the Hadamard Product, and that will give us the gradient wrt to wi,j(k)w_{i, j}^{(k)}wi,j(k)​.

Formulation of an Algorithm

The basic idea is, compute forward to get the values for each layer, and then starting from the back (the last layer), update the weights.
