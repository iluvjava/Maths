Intro

Welcome. Proximal gradient descend is a unconstrained optimization method, it aims to solve:

min⁡x{f(x)+h(x)}\min_x \{f(x) + h(x)\} xmin​{f(x)+h(x)}

where f(x),h(x)f(x), h(x)f(x),h(x) are convex but h(x)h(x)h(x) is non-smooth.

Intuitively

if f(x)f(x)f(x) is complicated, it's hard to solve, however, if we have the convexity information for function f(x)f(x)f(x) at a fixed point, then we might solve the following problem efficiently:

proxh,t(z)=arg⁡min⁡x{12t∥x−z∥2+h(x)}\underset{h, t}{\text{prox}}( z ) = \arg\min_x \left\lbrace \frac{1}{2t} \left\Vert x - z \right\Vert^2 + h(x) \right\rbrace h,tprox​(z)=argxmin​{2t1​∥x−z∥2+h(x)}

This is called the proximal operator, parameterized by a convexity information related parameter ttt, and the non-smooth function named h(x)h(x)h(x).
