How do we inverse matrix operations when we have the inverse of some kind of sum of matrices?

Results can be generalized to rings too.

List of Formulas

Let PPP be invertable squared matrix, then we can say that:

(I+P)−1=I−P（I+P)−1=I−(I+P)−1P(1)(I + P)^{-1} = I- P（I+ P)^{-1} = I - (I + P)^{-1}P \tag{1} (I+P)−1=I−P（I+P)−1=I−(I+P)−1P(1)

By symmetry

P(I+P)−1=(I+P)−1P(2)P(I + P)^{-1} = (I + P)^{-1}P \tag{2} P(I+P)−1=(I+P)−1P(2)

Let's get a big fancier:

(I+AB)−1=I−A(I−BA)−1B(3)(I + AB)^{-1} = I - A(I - BA)^{-1}B \tag{3} (I+AB)−1=I−A(I−BA)−1B(3)

Observe that ABABAB is swapped into BA. As long as this is invertible, this will be true, A,BA, BA,B doesn't have to be squared matrices.

Need to exploit both (1), (2), (3) for expression (4).

A(I+BA)−1=(I+AB)−1A(4)A(I + BA)^{-1} = (I + AB)^{-1}A \tag{4} A(I+BA)−1=(I+AB)−1A(4)

This is call the push through identity, just push the A through the ()−1()^{-1}()−1, and swapping the BABABA into ABABAB

(I+P)(I+P)−1=I(I+P)−1+P(I+P)−1=I(I+P)−1=I−P(I+P)−1\begin{aligned} (I + P)(I + P)^{-1} &= I \\ (I + P)^{-1} + P(I + P)^{-1} &= I \\ (I +P)^{-1} &= I - P(I + P)^{-1} \end{aligned} (I+P)(I+P)−1(I+P)−1+P(I+P)−1(I+P)−1​=I=I=I−P(I+P)−1​

Applications, Special Cases, Corollaries

Inverse of a Sum

(B+A)−1=B−1−B−1A(B+A)−1 (B + A)^{-1} = B^{-1} - B^{-1}A(B + A)^{-1} (B+A)−1=B−1−B−1A(B+A)−1

And symmetrically we also have:

(A+B)−1=A−1−A−1B(A+B)−1(A + B)^{-1} = A^{-1} - A^{-1}B(A + B)^{-1} (A+B)−1=A−1−A−1B(A+B)−1

For this one we are going to use expression (3) and the identity that for any matrix A,BA,BA,B such that ABABAB is invertible, we have:

AB=B−1A−1AB = B^{-1}A^{-1} AB=B−1A−1

Observe the following fact:

B−1(I+AB−1)−1=((I+AB−1)B)−1=(B+A)−1\begin{aligned} B^{-1}(I + AB^{-1})^{-1} = ((I + AB^{-1})B)^{-1} = (B + A)^{-1} \end{aligned} B−1(I+AB−1)−1=((I+AB−1)B)−1=(B+A)−1​

From expression (3), we can figure out (I+AB−1)(I + AB^{-1})(I+AB−1), which is going to be the following:

(I+AB−1)−1=(3)I−A(I+B−1A)−1B−1B−1(I+AB−1)=B−1−B−1A(I+B−1A)−1B−1=B−1−B−1A[B(I+B−1A)]−1(B+A)−1=B−1−B−1A(B+A)−1\begin{aligned} (I + AB^{-1})^{-1} &\underset{(3)}{=} I - A(I + B^{-1}A)^{-1}B^{-1} \\ B^{-1}(I + AB^{-1}) &= B^{-1} - B^{-1}A(I + B^{-1}A)^{-1}B^{-1} \\ &= B^{-1} - B^{-1}A[B(I + B^{-1}A)]^{-1} \\ (B + A)^{-1}&= B^{-1} - B^{-1}A(B + A)^{-1} \end{aligned} (I+AB−1)−1B−1(I+AB−1)(B+A)−1​(3)=​I−A(I+B−1A)−1B−1=B−1−B−1A(I+B−1A)−1B−1=B−1−B−1A[B(I+B−1A)]−1=B−1−B−1A(B+A)−1​

Note: Try apply this formula recursively, then we obtained matrix series.

Note: This can be used to derive the famous Woodburry's Formula.

■\blacksquare■
