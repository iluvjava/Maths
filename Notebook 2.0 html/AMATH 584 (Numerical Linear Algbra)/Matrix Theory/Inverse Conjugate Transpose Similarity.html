Theorem 2.1.9 in book <Matrix Analysis>

Intro

This is the theroem statement:

A∈MnA−1∼AH  ⟺  B∈Mn:A=B−1BHA\in \mathbb{M}_n \quad A^{-1} \sim A^H \iff B\in \mathbb{M}_n: A = B^{-1}B^H A∈Mn​A−1∼AH⟺B∈Mn​:A=B−1BH

The inverse of matrix AAA is similar to AHA^HAH if and only if AAA can be written as the product of B−1BHB^{-1}B^HB−1BH.

The assumptions are:

  • Matrix AAA is invertible.
  • Matrix BBB is invertible.
