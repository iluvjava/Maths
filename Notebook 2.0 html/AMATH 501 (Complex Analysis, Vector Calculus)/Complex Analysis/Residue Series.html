Intro

We can use the Residue theorem to figure out the infinite series.

Let's consider a really special function which is going to help us with that:

F(z):=πcot⁡(πz)f(z)=πf(z)cos⁡(πz)sin⁡(πz)F(z): = \pi \cot(\pi z)f(z) = \frac{\pi f(z)\cos(\pi z)}{\sin(\pi z)} F(z):=πcot(πz)f(z)=sin(πz)πf(z)cos(πz)​

  • ** Observations** The multiplier, the fraction part with: πcot⁡(πz)\pi\cot(\pi z)πcot(πz) has singularities on all the integers, and they are all simple poles. And when we design the contour, it's going to avoid the singularities of this one, and the function f(z)f(z)f(z).

Where f(x)f(x)f(x) can be any functions really, meromorphic preferably.

Now we are going to consider the following contour integral:

∮Cπcos⁡(πz)sin⁡(πz)f(z)  dz\oint_{C}\frac{\pi \cos(\pi z)}{\sin(\pi z)} f(z)\;dz ∮C​sin(πz)πcos(πz)​f(z)dz

Let's take a look at the Contour:

Vertices: ±(N+112)±i(N+12)\text{Vertices: } { \pm\left(N + 1\frac{1}{2}\right) \pm i\left(N + \frac{1}{2}\right) } Vertices: ±(N+121​)±i(N+21​)

And it's a square that goes counter-clock wise, avoiding all the singularities on the integers.

Now, let's figure out all the residues at the integers:

∑k∈ZRes(F(z),k)\sum_{k\in \mathbb{Z}} \text{Res}(F(z), k) k∈Z∑​Res(F(z),k)

And for each of the one, we have:

Because the singularities on the integer set are all simple poles, (And we did't show it here but it's true), we can use the limit theorem to find the value for the residue.

lim⁡z→n(z−n)πcos⁡(πz)f(z)∑n=1∞(−1)n−1((z−n)/π)2n−1(2n−1)!\lim_{z \rightarrow n} \frac{(z - n)\pi \cos(\pi z)f(z)} {\sum_{n = 1}^\infty (-1)^{n - 1}\frac{((z-n)/\pi)^{2n - 1}}{(2n-1)!}} z→nlim​∑n=1∞​(−1)n−1(2n−1)!((z−n)/π)2n−1​(z−n)πcos(πz)f(z)​

Where I put the cos⁡(πz)\cos(\pi z)cos(πz) into a series on the denominator, and then we will have:

lim⁡z→nπcos⁡(πz)f(z)∑n=1∞(−1)n−1((z−n))2n−2π2n−1(2n−1)!\lim_{z \rightarrow n} \frac{\pi \cos(\pi z)f(z)} {\sum_{n = 1}^\infty (-1)^{n - 1}\frac{((z-n))^{2n - 2}}{\pi^{2n - 1}(2n-1)!}} z→nlim​∑n=1∞​(−1)n−1π2n−1(2n−1)!((z−n))2n−2​πcos(πz)f(z)​

The z−nz - nz−n on the numerator canceled out with the z−nz-nz−n on the denominator, and then we also take out the π\piπ from the terms in the series.

And then at this point, we will be able to do a direct substitution for the z=nz = nz=n. which is going to give us:

(πcos⁡(πz)f(n))/π=f(n)(\pi\cos(\pi z)f(n))/\pi = f(n) (πcos(πz)f(n))/π=f(n)

Now, let's denote all the singularities of the function f(z)f(z)f(z) to be a set, name it SSS, then we can phrase the contour integrals in terms of singularities using the residue, giving us:

∮CF(z)dz=2πi∑n=−NNf(n)+2πi∑α∈SRes(F,α)\oint_{C}F(z) dz = 2\pi i\sum_{n=-N}^{N}f(n) + 2\pi i\sum_{\alpha \in S} \text{Res}(F, \alpha) ∮C​F(z)dz=2πin=−N∑N​f(n)+2πiα∈S∑​Res(F,α)

Now let's make an **additional assumption so that the outer contour integral will become zero as the size of the square approaches ∞\infty∞. the assumption is: **

lim⁡∣z∣→∞∣zf(z)∣=0\lim_{|z|\rightarrow \infty}|zf(z)| = 0 ∣z∣→∞lim​∣zf(z)∣=0

And then, we will be able to say that the contour integral will be zero as N→∞N\rightarrow \inftyN→∞, this is the case by considering the exponential form of cot⁡\cotcot which is going to give us:

cot⁡(z)=iexp⁡(iz)+exp⁡(−iz)exp⁡(iz)−exp⁡(−iz)\cot(z) = i\frac{\exp(iz) + \exp(-iz)}{\exp(iz) - \exp(-iz)} cot(z)=iexp(iz)−exp(−iz)exp(iz)+exp(−iz)​

Take note that, as ∣z∣→∞|z|\rightarrow \infty∣z∣→∞, this quotient is approaching 1, and this will imply that F(z)F(z)F(z) is approaching zero as the contour grows to infinity.

And then we have:

2πi∑n=−NNf(n)+2πi∑α∈SRes(F,α)=02\pi i\sum_{n=-N}^{N}f(n) + 2\pi i\sum_{\alpha \in S} \text{Res}(F, \alpha) = 0 2πin=−N∑N​f(n)+2πiα∈S∑​Res(F,α)=0

Which is giving us:

∑n=−∞∞f(n)=−∑α∈SRes(F,α)\sum_{n=-\infty}^{\infty}f(n) =- \sum_{\alpha \in S} \text{Res}(F, \alpha) n=−∞∑∞​f(n)=−α∈S∑​Res(F,α)

And this is fucking perfect.

Complications

What if the poles of f(z)f(z)f(z) and πcot⁡(πz)\pi\cot(\pi z)πcot(πz) overlapped with each other?

Example 1

Let's figure out the series:

∑n=−∞∞1n2+1\sum_{n = -\infty}^\infty \frac{1}{n^2 + 1} n=−∞∑∞​n2+11​

Take notice of the recurrence relationship:

∑n=−∞∞1n2+1=1+2∑n=1∞1n2+1\sum_{n=-\infty}^{\infty}\frac{1}{n^2 + 1} = 1 + 2\sum_{n = 1}^{\infty} \frac{1}{n^2 + 1} n=−∞∑∞​n2+11​=1+2n=1∑∞​n2+11​

We can use the recurrence to get some discount for our infinite series.

Therefore, the residue of the function F(z)F(z)F(z) will be evaluated via the form:

Res(F,±i)=lim⁡z→±i(z∓i)πcot⁡(πz)(z−i)(z+i)=lim⁡z→±iπcot⁡(πz)(z±i)\text{Res}(F, \pm i) = \lim_{z\rightarrow \pm i} (z \mp i)\frac{\pi\cot(\pi z)}{(z - i)(z + i)} =\lim_{z\rightarrow \pm i} \frac{\pi\cot(\pi z)}{(z\pm i)} Res(F,±i)=z→±ilim​(z∓i)(z−i)(z+i)πcot(πz)​=z→±ilim​(z±i)πcot(πz)​

And then we have:

Res(F,±i)=−π2coth⁡(π)\text{Res}(F, \pm i) = -\frac{\pi}{2}\coth(\pi) Res(F,±i)=−2π​coth(π)

And then we can use the Exponential form of the cot⁡\cotcot to cancel out the imaginary unit at the denominator, and take not that the cot⁡\cotcot function is odd, so the Operator Superposition canceled out.

Therefore, the infinite series has the representation that:

∑n=−∞∞1n2+1=πcoth⁡(π)\sum_{n=-\infty}^\infty \frac{1}{n^2 + 1} =\pi \coth(\pi) n=−∞∑∞​n2+11​=πcoth(π)

And then we get another one for free using the relation:

∑n=1∞1n2+1=12(πcoth⁡(π)−1)\sum_{n = 1}^{\infty} \frac{1}{n^2 + 1} = \frac{1}{2}(\pi\coth(\pi) - 1) n=1∑∞​n2+11​=21​(πcoth(π)−1)
