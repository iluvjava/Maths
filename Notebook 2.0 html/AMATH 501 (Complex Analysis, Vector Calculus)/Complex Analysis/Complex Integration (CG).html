Intro

Integration on complex functions are line integral.

Let γ\gammaγ be the line between 2 points, z0,z1z_0, z_1z0​,z1​. And let's parameterize a curve that links between these 2 paths, and it will be like the following:

And then the integral of on the complex plane will be the following:

∫z0z1zdz=∫x0z1z(t)z′(t)dt\int_{z_0}^{z_1} zdz = \int_{x_0}^{z_1}z(t)z'(t)dt ∫z0​z1​​zdz=∫x0​z1​​z(t)z′(t)dt

This is the starting point.

Comment:

A smooth curve means that the derivative of the curve is continuous, and the derivative at the end point t=at = at=a, or t=bt = bt=b cannot be zero.

A piece-wise smooth curve means that: It's joined together by finitely many smooth curve, end to end, no space in between.

The above theorem is applicable for both smooth and piece-wise smooth curve.

Fundamental Theorem of Contour Integration (FTCI)

If the function is analytic, then ∫ab=F(b)−F(b)\int_{a}^{b} = F(b) - F(b)∫ab​=F(b)−F(b)

Notice that, this is the same as the Fundamental Theorem of Calculus.

This is the case because f(z)=u(x,y)+iv(x,y)f(z) = u(x, y) + iv(x, y)f(z)=u(x,y)+iv(x,y) can be viewed as 2 functions:

Re(f(z))=u(x,y)Im(f(z))=v(x,y)\text{Re}(f(z)) = u(x,y) \quad \text{Im}(f(z)) = v(x, y) Re(f(z))=u(x,y)Im(f(z))=v(x,y)

Ok, the proof here is obvious, if we do enough algebra that is, the line integral on this complex function can be implicitly stated as the following:

∫z0z1(u(x,y)+iv(x,y))(x′+iy′)dt\int_{z_0}^{z_1} (u(x, y) + i v(x, y))(x' + iy')dt ∫z0​z1​​(u(x,y)+iv(x,y))(x′+iy′)dt

∫z0z1(u(x,y)x′−v(x,y)y′)+i(v(x,y)+u(x,y)y′)dt\int_{z_0}^{z_1} (u(x, y)x' - v(x, y)y') + i(v(x, y) + u(x, y)y')dt ∫z0​z1​​(u(x,y)x′−v(x,y)y′)+i(v(x,y)+u(x,y)y′)dt

Observe that, the vector field: (u,−v)(u, -v)(u,−v) has a curl of −uy−vx-u_y - v_x−uy​−vx​, and by Cauchy Riemann conditions, this is zero.

Similarly, the vector field (v,u)(v, u)(v,u) has a curl of ∂xu−∂yv\partial_x u - \partial_y v∂x​u−∂y​v which is zero according to Cauchy Riemann Conditions.

Then, both line integrals on the Real and Imaginary part of the function will be a conservative field, and in that case, we know that:

∫x0x1zdz=F(z1)−F(x2)\int_{x_0}^{x_1} zdz = F(z_1) - F(x_2) ∫x0​x1​​zdz=F(z1​)−F(x2​)

Cauchy Goursat Theorem

If f(z)f(z)f(z) is analytic inside and on a simple closed curve C, then the integral around the closed C is zero.

This is followed from the FTCI theorem, and definitely true because of that Conservative Field Argument.

Note: 1/z1/z1/z is not applicable for Cauchy Goursat Theorem if the closed path include the origin because the function is not differentiable at the origin, and this breaks the preconditions of the theorem.

If f(z)f(z)f(z) is analytic between C1 and C2, then the line integral on C1 and C2 gives the same value.
