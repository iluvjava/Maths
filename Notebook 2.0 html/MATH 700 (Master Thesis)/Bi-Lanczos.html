Intro

The Bi-conjugate Gradient algorithm uses the 2-side Lanczos algorithm for non-symmetric matrix, constructing 2 arthogonal subsapces during the iterations.

(span←col)(Vk)=Kk(A,r0)(span←col)(Wk)=Kk(AH,r^0)\begin{aligned} \left( \text{span} \leftarrow \text{col} \right)(V_k) = \text{K}_k(A, r_0) \\ (\text{span}\leftarrow \text{col})(W_k) = \text{K}_k(A^H, \hat{r}_0) \end{aligned} (span←col)(Vk​)=Kk​(A,r0​)(span←col)(Wk​)=Kk​(AH,r^0​)​

Such that:

WkTVk=IW_k^TV_k = I WkT​Vk​=I

Serious Breakdown

The algorithm breaks down when during the iterations jjj, we have:

vj,wj≠0∧⟨vj,wj⟩=0v_j, w_j \neq 0 \wedge \langle v_j, w_j\rangle = 0 vj​,wj​=0∧⟨vj​,wj​⟩=0

When this happens, βj=0\beta_j = 0βj​=0, whic later, it's used as the Denominator, and then we will have a divides by zero problem.
