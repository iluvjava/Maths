[[AMATH 582/SVM]], [[PCA (Principal Components Analysis)]]

Here we talk about another extension of the PCA method. The Kernel PCA which performs the PCA at a non-linear subspace compare to the original dimension where the data lies in. The same kernel trick used for SVM.

Reference: Link, and wikipedia

In practice, people can use Kernel PCA from the sklearn library.

Intro

When the features are not necessarily separable under the linear subspace, we will use the kernel tricks to get a better separations in a new dimension.

Note: The assumption is not the same as the low-rank assumption for the data set.

Linear PCA

When computing the singular value decomposition for XXX, one of the approach is to find the Co-variance matrix for XXX.

VΛVT=XXTUΩUT=XTXV\Lambda V^T = XX^T \quad U\Omega U^T = X^TX VΛVT=XXTUΩUT=XTX

This is feasible because symmetric matrix (Or Hermitian) has orthogonal Eigenvectors, and in addition, the non-zero diagonals of the matrix Λ,Ω\Lambda, \OmegaΛ,Ω are having the same non-zero entries, and they the singular values of the matrix XXX.[1]

And the principal components, are the VVV matrix and the profile matrix is the UUU matrix.

And the focus here is that, the idea of an inner product matrix, or the co-variance matrix can be generalized here, just like the kernel trick used for the SVM model.

Here is a note from wikipedia, about the larger over all theme of the kernel PCA method:

Because we are never working directly in the feature space, the kernel-formulation of PCA is restricted in that it computes not the principal components themselves, but the projections of our data onto those components

Kernel PCA

The key idea is to simplify and use the kernel function K(xi,xi)K(x_i, x_i)K(xi​,xi​) for the given function ϕ\phiϕ to describe the kernel PCA.

Consider a mapping function from the linear feature space to the non-linear feature space:

xi↦ϕ(xi)x_i \mapsto \phi(x_i) xi​↦ϕ(xi​)

Where, xix_ixi​ is one of the samples in the feature space.

The kernel PCA will be treating the non-linear space as the new feature space and then apply PCA algorithm in here instead.

First assume that the data is:

μ=1n∑i=1nϕ(xi)\mu = \frac{1}{n}\sum_{i = 1}^{n}\phi(x_i) μ=n1​i=1∑n​ϕ(xi​)

The covariance in the non-linear space is given by:

C=1n∑i=1nϕ(xi)ϕ(xi)TC = \frac{1}{n}\sum_{i = 1}^{n}\phi(x_i)\phi(x_i)^T C=n1​i=1∑n​ϕ(xi​)ϕ(xi​)T

Where: the matrix is described by a rank one outter produce of the features under the new non-linear space.

Then the eigenvector of the co-variance matrix is given by:

Cv=λvCv = \lambda v Cv=λv

The eigenvector is a linear combination of all the features, giving us:

v=∑i=1nαiϕ(xi)v = \sum_{i = 1}^{n}\alpha_i\phi(x_i) v=i=1∑n​αi​ϕ(xi​)

The vector v exists in the non-linear subspace, in the alternative basis.

Then the product CvCvCv is:

Cv=1n∑i=1nϕ(xi)ϕ(xi)Tv=λv(1)Cv = \frac{1}{n}\sum_{i = 1}^{n}\phi(x_i)\phi(x_i)^Tv = \lambda v \tag{1} Cv=n1​i=1∑n​ϕ(xi​)ϕ(xi​)Tv=λv(1)

assuming that λ≠0\lambda\neq 0λ=0, then the quantity vvv will be represented as:

v=1λn∑i=1nϕ(xi)ϕ(xi)Tv(2)v = \frac{1}{\lambda n}\sum_{i = 1}^{n} \phi(x_i)\phi(x_i)^Tv \tag{2} v=λn1​i=1∑n​ϕ(xi​)ϕ(xi​)Tv(2)

The eigenvector of the covariance matrix is now represented as a linear combination of the kernel vector. Next we want the matrix to be expressed as a linear combination of ϕ(xi)T\phi(x_i)^Tϕ(xi​)T.

ϕ(xi)ϕ(xi)Tv=ϕ(xi)(vTϕ(xi))T=vTϕ(xi)ϕT(xi)\phi(x_i)\phi(x_i)^Tv = \phi(x_i)(v^T\phi(x_i))^T = v^T\phi(x_i)\phi^T(x_i) ϕ(xi​)ϕ(xi​)Tv=ϕ(xi​)(vTϕ(xi​))T=vTϕ(xi​)ϕT(xi​)

Then we have:

v=∑i=1nvTϕ(xi)ϕ(xi)Tαi=vTϕ(xi)v = \sum_{i = 1}^{n} v^T\phi(x_i)\phi(x_i)^T \quad \alpha_i = v^T\phi(x_i) v=i=1∑n​vTϕ(xi​)ϕ(xi​)Tαi​=vTϕ(xi​)

Substitute expression (2) into expression (1), then:

Cv=1n∑i=1nϕ(xi)ϕ(xi)T∑l=1nαi,lϕ(xl)Cv = \frac{1}{n} \sum_{i = 1}^{n} \phi(x_i)\phi(x_i)^T \sum_{l = 1}^{n} \alpha_{i, l}\phi(x_l) Cv=n1​i=1∑n​ϕ(xi​)ϕ(xi​)Tl=1∑n​αi,l​ϕ(xl​)

1n∑i=1nϕ(xi)∑l=1nαi,lϕ(xl)ϕ(xi)T\frac{1}{n} \sum_{i = 1}^{n} \phi(x_i) \sum_{l = 1}^{n} \alpha_{i, l}\phi(x_l)\phi(x_i)^T n1​i=1∑n​ϕ(xi​)l=1∑n​αi,l​ϕ(xl​)ϕ(xi​)T

1n∑i=1nϕ(xi)∑l=1nαi,lK(xl,xi)\frac{1}{n} \sum_{i = 1}^{n} \phi(x_i) \sum_{l = 1}^{n} \alpha_{i, l}K(x_l, x_i) n1​i=1∑n​ϕ(xi​)l=1∑n​αi,l​K(xl​,xi​)

Now, we choose any sample's non-linear representation, say: ϕ(xk)\phi(x_k)ϕ(xk​), then we have:

1n∑i=1nϕ(xk)ϕ(xi)∑l=1nαi,lK(xl,xi)=1n∑i=1nK(xk,xi)∑l=1nαi,lK(xl,xi)(3)\frac{1}{n} \sum_{i = 1}^{n} \phi(x_k) \phi(x_i) \sum_{l = 1}^{n} \alpha_{i, l}K(x_l, x_i) = \frac{1}{n} \sum_{i = 1}^{n} K(x_k, x_i) \sum_{l = 1}^{n} \alpha_{i, l}K(x_l, x_i) \tag{3} n1​i=1∑n​ϕ(xk​)ϕ(xi​)l=1∑n​αi,l​K(xl​,xi​)=n1​i=1∑n​K(xk​,xi​)l=1∑n​αi,l​K(xl​,xi​)(3)

Now, let the matrix K\mathbf{K}K all the inner products K(xi,xj)K(x_i, x_j)K(xi​,xj​) packed together, then we have:

K=[K(x1,x1)K(x2,x1)⋯K(x1,xn)K(x2,x1)K(x2,x2)⋯K(x2,xn)⋮K(xn,x1)K(xn,x2)⋯K(xn,xn)]=KT\mathbf{K} = \begin{bmatrix} K(x_1, x_1) & K(x_2, x_1) & \cdots & K(x_1, x_n) \\ K(x_2, x_1) & K(x_2, x_2) & \cdots & K(x_2, x_n) \\ \vdots & & & \\ K(x_n, x_1) & K(x_n, x_2) & \cdots & K(x_n, x_n) \end{bmatrix} = \mathbf{K}^T K=⎣⎡​K(x1​,x1​)K(x2​,x1​)⋮K(xn​,x1​)​K(x2​,x1​)K(x2​,x2​)K(xn​,x2​)​⋯⋯⋯​K(x1​,xn​)K(x2​,xn​)K(xn​,xn​)​⎦⎤​=KT

The matrix is assumed to be symmetric.

Then, we can reconsider the expression above, with this matrix:

(Kαj)j=∑l=1nαj,lK(xl,xj)(\mathbf{K}\alpha_j)_j = \sum_{l = 1}^{n} \alpha_{j, l}K(x_l, x_j) (Kαj​)j​=l=1∑n​αj,l​K(xl​,xj​)

Now, go back to expression (3), choose k=1,2,3⋯nk = 1, 2, 3\cdots nk=1,2,3⋯n, and then stack then vertically together, we will have the expression in terms of the Kernel Matrix, in the form as:

Cv=1nK2αj(4)Cv = \frac{1}{n}\mathbf{K}^2\alpha_j \tag{4} Cv=n1​K2αj​(4)

Take another branch from the expression for CvCvCv, and then we know that:

Cv=λv=λ∑i=1nαiϕ(xi)Cv = \lambda v = \lambda \sum_{i = 1}^{n}\alpha_i\phi(x_i) Cv=λv=λi=1∑n​αi​ϕ(xi​)

ϕ(xk)Cv=λ∑i=1nαiϕ(xk)Tϕ(xi)=λ∑i=1nαiK(xk,xi)=λ(Kα)k\phi(x_k)Cv = \lambda \sum_{i = 1}^{n} \alpha_i\phi(x_k)^T\phi(x_i) = \lambda \sum_{i = 1}^{n} \alpha_iK(x_k, x_i) = \lambda (\mathbf{K}\alpha)_k ϕ(xk​)Cv=λi=1∑n​αi​ϕ(xk​)Tϕ(xi​)=λi=1∑n​αi​K(xk​,xi​)=λ(Kα)k​

However, we are going to add a subscript for α\alphaα, and name it αj\alpha_jαj​, so it's consistent with expression (4). Combining the expression above with expression (4) we have:

1nK2αj=λKαj  ⟹  (a)Kαj=nλjαj(5)\frac{1}{n}\mathbf{K}^2\alpha_j = \lambda\mathbf{K}\alpha_j \underset{(a)}{\implies} \mathbf{K} \alpha_j = n\lambda_j\alpha_j \tag{5} n1​K2αj​=λKαj​(a)⟹​Kαj​=nλj​αj​(5)

Principal Component Projection:

For any new point, xxx, it's projection onto the principal component will be:

ϕ(x)Tvj=∑i=1nαi,jϕ(x)Tϕ(xi)=∑i=1nαi,jK(x,xi)\phi(x)^Tv_j = \sum_{i = 1}^{n}\alpha_{i, j}\phi(x)^T\phi(x_i) = \sum_{i = 1}^{n}\alpha_{i, j}K(x, x_i) ϕ(x)Tvj​=i=1∑n​αi,j​ϕ(x)Tϕ(xi​)=i=1∑n​αi,j​K(x,xi​)
